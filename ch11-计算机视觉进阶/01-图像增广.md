# 11.1 图像增广

## 学习目标

- 理解图像增广的目的和作用
- 掌握常见的增广方法
- 了解如何选择合适的增广策略

## 核心概念

### 什么是图像增广？

**图像增广就是"变着花样"扩充训练数据。**

一张猫的照片，可以：
- 水平翻转 → 得到镜像猫
- 随机裁剪 → 得到不同角度的猫
- 调整亮度 → 得到白天/晚上的猫
- 旋转角度 → 得到歪头的猫

**这些变换后的图片，本质上还是"猫"，但看起来不同。**

### 为什么需要图像增广？

| 问题 | 图像增广的解决方案 |
|------|-------------------|
| 数据量少 | 一张图片变多张，扩充数据 |
| 模型过拟合 | 让模型看到更多样化的数据 |
| 泛化能力弱 | 模型学会应对各种变化 |
| 实际场景复杂 | 模拟真实世界的各种情况 |

**类比：考试前刷题。**

只做一道题 → 可能只会这一题的解法
做这道题的多种变体 → 学会这类题的通用解法

### 常见的增广方法

| 方法 | 效果 | 适用场景 |
|------|------|----------|
| 水平翻转 | 镜像翻转 | 自然图像（猫、狗） |
| 垂直翻转 |上下翻转 | 一般不用（猫不能倒着站） |
| 随机裁剪 | 截取局部 | 物体识别 |
| 旋转 | 角度变换 | 文字、医疗图像 |
| 颜色抖动 | 亮度/对比度变化 | 应对不同光照 |
| 添加噪声 | 模拟传感器噪声 | 低质量图像处理 |

## 原理解析

### 几何变换

**翻转：**
- 水平翻转：左右镜像（`img[:, ::-1]`）
- 垂直翻转：上下镜像（`img[::-1, :]`）

**裁剪：**
- 随机选择裁剪区域
- 保留物体关键部分
- 让模型学会从不同角度识别物体

**旋转：**
- 绕中心点旋转指定角度
- 可能超出边界，需要填充

### 颜色变换

**亮度调整：**
```
新像素 = 原像素 + 增量
```

**对比度调整：**
```
新像素 = (原像素 - 平均值) ×因子 + 平均值
```

**颜色抖动：**
同时随机调整亮度、对比度、饱和度。

### 增广的注意事项

1. **保持标签正确**：翻转图片后，标签还是"猫"，但如果翻转文字图片，文字可能就变了。

2. **不要过度增广**：增广太厉害可能把"猫"变成不像"猫"，反而有害。

3. **验证集不增广**：验证集要反映真实情况，不应该用增广数据。

## 代码实战

### 基础增广方法

```python
import numpy as np

def horizontal_flip(img):
    """水平翻转"""
    return img[:, ::-1]

def vertical_flip(img):
    """垂直翻转"""
    return img[::-1, :]

def random_crop(img, crop_size):
    """随机裁剪"""
    h, w = img.shape[:2]
    ch, cw = crop_size

    # 随机选择裁剪起点
    top = np.random.randint(0, h - ch)
    left = np.random.randint(0, w - cw)

    return img[top:top+ch, left:left+cw]

def adjust_brightness(img, factor):
    """调整亮度"""
    return np.clip(img * factor, 0, 255).astype(np.uint8)

def add_noise(img, noise_level=10):
    """添加随机噪声"""
    noise = np.random.randn(*img.shape) * noise_level
    return np.clip(img + noise, 0, 255).astype(np.uint8)


# === 演示 ===
np.random.seed(42)

# 创建模拟图像（8x8 RGB）
img = np.random.randint(0, 255, (8, 8, 3), dtype=np.uint8)

print("=== 图像增广演示 ===")
print(f"原始图像形状: {img.shape}")

print("\n各增广方法:")
print(f"水平翻转: 形状不变，内容左右镜像")
print(f"垂直翻转: 形状不变，内容上下镜像")
print(f"随机裁剪: 形状变小，截取局部")
print(f"亮度调整: 像素值整体乘以因子")
print(f"添加噪声: 每个像素随机加减值")

# 演示裁剪
cropped = random_crop(img, (4, 4))
print(f"\n裁剪后形状: {cropped.shape}")

# 演示亮度
bright = adjust_brightness(img, 1.5)
dark = adjust_brightness(img, 0.5)
print(f"亮度增强: 平均像素值{img.mean():.1f} → {bright.mean():.1f}")
print(f"亮度降低: 平均像素值{img.mean():.1f} → {dark.mean():.1f}")

# 演示噪声
noisy = add_noise(img, 20)
print(f"添加噪声: 像素变化范围约 ±20")
```

## 避坑指南

1. **标签也要跟着变**：目标检测中，翻转图片后，边界框坐标也要相应调整。

2. **某些任务不能翻转**：文字识别、医疗图像（心脏位置固定）等。

3. **过度增广反而有害**：比如把猫旋转 180 度，看起来不像猫了。

4. **测试集不要增广**：测试集要评估真实场景的表现。
