# 11.5 风格迁移

## 学习目标

- 理解风格迁移的原理
- 掌握内容损失和风格损失的概念
- 了解格拉姆矩阵的作用

## 核心概念

### 风格迁移：把照片变成艺术品

**风格迁移：用一张图的"风格"来画另一张图的"内容"。**

| 输入 | 例子 |
|------|------|
| 内容图 | 你拍的照片 |
| 风格图 | 梵高的《星空》 |
| 输出 | 星空风格的照片 |

风格迁移让普通照片变成艺术作品。

### 核心思路

**内容**：照片中的物体、轮廓、布局
**风格**：颜色、纹理、笔触、艺术感

风格迁移的目标：
- 保持内容图的"内容"（物体不变）
- 应用风格图的"风格"（变成星空画风）

### 损失函数的设计

需要两个损失：

| 损失 | 作用 | 计算方式 |
|------|------|----------|
| 内容损失 | 保持内容不变 | 内容图和生成图的特征差异 |
| 风格损失 | 学习风格 | 风格图和生成图的风格特征差异 |

总损失 = 内容损失 + 权重 × 风格损失

## 原理解析

### 内容损失

用预训练 CNN（如 VGG）提取特征：

```
内容图 → VGG → 特征 F_content
生成图 → VGG → 特征 F_generated

内容损失 = ||F_content - F_generated||²
```

特征相似，内容就相似。

### 格拉姆矩阵：捕捉风格

风格 = 特征之间的"相关性"。

比如梵高的风格：
- 红色和黄色经常一起出现（高相关）
- 某种笔触模式反复出现

**格拉姆矩阵（Gram Matrix）** 计算特征之间的相关性：

```
Gram(F) = F × F^T
```

Gram 矩阵的每个元素表示两个特征通道的相关程度。

### 风格损失

```
风格图 → VGG → Gram(F_style)
生成图 → VGG → Gram(F_generated)

风格损失 = ||Gram(F_style) - Gram(F_generated)||²
```

Gram 矩阵相似，风格就相似。

### 训练过程

风格迁移的训练比较特殊：

```
1. 固定 VGG 网络（只用它提取特征）
2. 初始化生成图（可以是内容图的副本）
3. 计算内容损失和风格损失
4. 更新生成图（不是更新网络，而是更新图片！）
5. 重复直到满意
```

**关键点：优化的是生成图，不是网络参数。**

## 代码实战

### 格拉姆矩阵计算

```python
import numpy as np


def gram_matrix(features):
    """计算格拉姆矩阵

    Args:
        features: 特征图，形状 (h, w, c) 或 (c, h, w)

    Returns:
        gram: 格拉姆矩阵，形状 (c, c)
    """
    # 假设形状是 (h, w, c)
    h, w, c = features.shape

    # 展平空间维度
    features_flat = features.reshape(h * w, c)

    # 计算格拉姆矩阵: F^T × F
    gram = np.dot(features_flat.T, features_flat) / (h * w)

    return gram


def content_loss(content_features, generated_features):
    """内容损失

    Args:
        content_features: 内容图特征
        generated_features: 生成图特征

    Returns:
        loss: 内容损失值
    """
    return np.mean((content_features - generated_features) ** 2)


def style_loss(style_gram, generated_gram):
    """风格损失

    Args:
        style_gram: 风格图的格拉姆矩阵
        generated_gram: 生成图的格拉姆矩阵

    Returns:
        loss: 风格损失值
    """
    return np.mean((style_gram - generated_gram) ** 2)


def total_loss(content_loss, style_loss, style_weight=1000):
    """总损失

    Args:
        content_loss: 内容损失
        style_loss: 风格损失
        style_weight: 风格损失权重

    Returns:
        loss: 总损失
    """
    return content_loss + style_weight * style_loss


# === 演示 ===
print("=== 风格迁移基础演示 ===")

np.random.seed(42)

# 模拟特征图
# 假设 VGG 提取的特征是 (8, 8, 16)
feature_h, feature_w, feature_c = 8, 8, 16

content_features = np.random.randn(feature_h, feature_w, feature_c)
style_features = np.random.randn(feature_h, feature_w, feature_c)
generated_features = np.random.randn(feature_h, feature_w, feature_c)

print(f"特征图形状: ({feature_h}, {feature_w}, {feature_c})")

# 计算格拉姆矩阵
content_gram = gram_matrix(content_features)
style_gram = gram_matrix(style_features)
generated_gram = gram_matrix(generated_features)

print(f"\n格拉姆矩阵形状: {content_gram.shape}")
print(f"格拉姆矩阵表示特征通道之间的相关性")

# 计算损失
c_loss = content_loss(content_features, generated_features)
s_loss = style_loss(style_gram, generated_gram)
t_loss = total_loss(c_loss, s_loss, style_weight=1000)

print(f"\n=== 损失计算 ===")
print(f"内容损失: {c_loss:.4f}")
print(f"风格损失: {s_loss:.4f}")
print(f"总损失 (风格权重=1000): {t_loss:.4f}")

print("\n=== 风格迁移流程 ===")
print("1. 固定 VGG 网络（用于提取特征）")
print("2. 初始化生成图")
print("3. 提取三张图的特征：内容图、风格图、生成图")
print("4. 计算内容损失（特征差异）")
print("5. 计算风格损失（格拉姆矩阵差异）")
print("6. 更新生成图像素（不是网络参数）")
print("7. 重复直到满意")

print("\n=== 关键概念 ===")
print("内容损失: 保持物体形状和布局")
print("风格损失: 学习颜色纹理和笔触")
print("格拉姆矩阵: 捕捉特征之间的相关性")
print("风格权重: 控制风格化程度")
```

## 避坑指南

1. **风格权重过大**：生成的图可能完全变成风格，失去内容。

2. **只用一层特征**：风格应该从多层提取，否则风格不够丰富。

3. **训练时间不足**：风格迁移需要多次迭代才能看到效果。

4. **图片分辨率问题**：高分辨率图片需要更多计算，可以先处理低分辨率版本。
