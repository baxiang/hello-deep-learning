# 11.4 语义分割

## 学习目标

- 理解语义分割的任务定义
- 掌握转置卷积的概念
- 了解全卷积网络的结构

## 核心概念

### 语义分割：给每个像素分类

**图像分类**：整张图一个标签（"猫"）
**目标检测**：框出物体位置（"猫，在左上角"）
**语义分割**：每个像素一个标签（"这只猫的每个像素都是'猫'"）

语义分割输出一张"分割图"，每个像素表示类别：
- 猫的像素 → 标记为"猫"
- 背景像素 → 标记为"背景"
- 狗的像素 → 标记为"狗"

**类比：涂色书。**

图像分类 → 说这本书是"动物主题"
目标检测 → 用框圈出每个动物
语义分割 → 把每只动物涂上不同颜色

### 全卷积网络（FCN）

传统 CNN 有全连接层，输出固定大小的向量。
语义分割需要输出和输入同样大小的图。

**全卷积网络：把全连接层换成卷积层。**

- 输入：任意大小的图像
- 输出：同样大小的分割图

### 转置卷积：从粗到精

CNN 通过卷积和池化，图像越来越小（下采样）。
语义分割需要恢复原始大小（上采样）。

**转置卷积（Transposed Convolution）就是"反向的卷积"。**

普通卷积：大图 → 小图
转置卷积：小图 → 大图

## 原理解析

### 下采样和上采样

**下采样：**
```
卷积 → 池化 → 卷积 → 池化 → ...
图像越来越小，特征越来越抽象
```

**上采样：**
```
转置卷积 → 转置卷积 → ...→ 输出分割图
图像越来越大，恢复空间分辨率
```

### 转置卷积的工作原理

普通卷积：
```
输入 4x4 → 卷积核 3x3，步长 1 → 输出 2x2
```

转置卷积（反向）：
```
输入 2x2 → 转置卷积核 3x3，步长 1 → 输出 4x4
```

原理：
- 每个输入像素"扩散"到多个输出位置
- 卷积核决定扩散的方式
- 重叠位置累加

### FCN 的结构

```
输入图像
  ↓
VGG 特征提取（下采样）
  ↓
特征图（较小）
  ↓
1x1 卷积（代替全连接）
  ↓
转置卷积（上采样）
  ↓
分割图（恢复原始大小）
```

**跳跃连接**：把浅层的细节信息和深层融合，得到更精确的分割。

## 代码实战

### 转置卷积示意

```python
import numpy as np


def simple_transposed_conv(input_data, kernel, stride=2):
    """简化的转置卷积演示

    Args:
        input_data: 输入数组，形状 (h, w)
        kernel: 卷积核，形状 (kh, kw)
        stride: 步长

    Returns:
        output: 上采样后的数组
    """
    h, w = input_data.shape
    kh, kw = kernel.shape

    # 输出大小
    out_h = (h - 1) * stride + kh
    out_w = (w - 1) * stride + kw

    output = np.zeros((out_h, out_w))

    # 每个"输入像素"扩展到输出
    for i in range(h):
        for j in range(w):
            # 输入像素扩展的起始位置
            out_i = i * stride
            out_j = j * stride

            # 卷积核作用
            output[out_i:out_i+kh, out_j:out_j+kw] += input_data[i, j] * kernel

    return output


def upsampling_bilinear(input_data, scale=2):
    """双线性插值上采样

    Args:
        input_data: 输入数组
        scale: 放大倍数

    Returns:
        output: 上采样后的数组
    """
    h, w = input_data.shape
    new_h, new_w = h * scale, w * scale

    output = np.zeros((new_h, new_w))

    for i in range(new_h):
        for j in range(new_w):
            # 对应原输入的位置
            src_i = i / scale
            src_j = j / scale

            # 四个相邻点
            i0, j0 = int(src_i), int(src_j)
            i1 = min(i0 + 1, h - 1)
            j1 = min(j0 + 1, w - 1)

            # 插值权重
            di, dj = src_i - i0, src_j - j0

            # 双线性插值
            output[i, j] = (1 - di) * (1 - dj) * input_data[i0, j0] + \
                           (1 - di) * dj * input_data[i0, j1] + \
                           di * (1 - dj) * input_data[i1, j0] + \
                           di * dj * input_data[i1, j1]

    return output


# === 演示 ===
print("=== 语义分割基础演示 ===")

# 模拟特征图（2x2）
feature_map = np.array([[1, 2],
                        [3, 4]])

print("输入特征图 (2x2):")
print(feature_map)

# 转置卷积核
kernel = np.array([[1, 1],
                   [1, 1]])

# 转置卷积上采样
transposed_output = simple_transposed_conv(feature_map, kernel, stride=2)

print("\n转置卷积输出 (stride=2):")
print(transposed_output)
print(f"输出形状: {transposed_output.shape}")

# 双线性插值上采样
bilinear_output = upsampling_bilinear(feature_map, scale=2)

print("\n双线性插值输出:")
print(bilinear_output)
print(f"输出形状: {bilinear_output.shape}")

print("\n=== 上采样方法对比 ===")
print("转置卷积:")
print("  - 可学习的上采样方式")
print("  - 参数可以优化")
print("  - 可能产生棋盘格效应")

print("\n双线性插值:")
print("  - 固定的插值方式")
print("  - 无参数")
print("  - 输出平滑")

print("\n=== 语义分割关键点 ===")
print("1. 每个像素都要分类")
print("2. 全卷积网络：没有全连接层")
print("3. 转置卷积：从粗到精上采样")
print("4. 跳跃连接：融合多层信息")
```

## 避坑指南

1. **转置卷积的棋盘格效应**：不当的参数会产生网格状噪声，需要调整卷积核大小和步长。

2. **输出尺寸计算错误**：转置卷积的输出大小取决于卷积核大小、步长和填充。

3. **类别不平衡**：背景像素通常比物体像素多，需要调整损失函数权重。
