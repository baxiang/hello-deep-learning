## 学习目标

- 理解目标检测的任务定义

- 了解 IoU 的计算方法

## 核心概念

### 目标检测：找出图中的物体

**图像分类**：回答"图里是什么？"（猫）
**目标检测**：回答"图里有什么？在哪里？"（猫，在左上角）

目标检测输出：
- **类别**：是什么物体
- **位置**：在图像的哪个区域（用边界框表示）

### 边界框（Bounding Box）

边界框用矩形框出物体的位置，通常用 4 个数值表示：

| 格式 | 含义 |
|------|------|
| (xmin, ymin, xmax, ymax) | 左上角和右下角坐标 |
| (x_center, y_center, width, height) | 中心点和宽高 |

### 锚框（Anchor Box）

目标检测需要"猜测"物体可能的位置和大小。

**锚框就是预设的"候选框"。**

比如设置几种典型大小的锚框：
- 小框：适合检测小物体（手机）
- 中框：适合检测中等物体（人）
- 大框：适合检测大物体（汽车）

模型会基于这些锚框进行调整，得到精确的边界框。

### IoU：衡量框的重合程度

**IoU（Intersection over Union）** = 两个框的交集面积 / 并集面积

```
IoU = 交集面积 / 并集面积
```

- IoU = 1：两个框完全重合
- IoU = 0.5：一半重叠
- IoU = 0：完全不重叠

IoU 用于：
- 判断预测框是否准确（IoU > 0.5 通常认为检测成功）
- 训练时计算损失

## 原理解析

### 边界框的表示

**角点格式：**
```
xmin, ymin: 左上角坐标
xmax, ymax: 右下角坐标
```

**中心点格式：**
```
x_center: 中心x 坐标
y_center: 中心 y 坐标
width: 框宽度
height: 框高度
```

两者可以互相转换。

### IoU 的计算

```
交集宽度 = min(xmax1, xmax2) - max(xmin1, xmin2)
交集高度 = min(ymax1, ymax2) - max(ymin1, ymin2)
交集面积 = 交集宽度 × 交集高度（如果 > 0）

并集面积 = 面积1 + 面积2 - 交集面积

IoU = 交集面积 / 并集面积
```

### 锚框的作用

模型不可能预测任意位置和大小的框，计算量太大。

锚框的设计思路：
1. 预设一组典型框（锚框）
2. 模型预测每个锚框的"调整量"
3. 用调整量修正锚框，得到最终预测框

这样模型只需要学习"如何调整"，而不是"直接预测位置"。

## 代码实战

### IoU 计算

```python
import numpy as np


def calculate_iou(box1, box2):
    """计算两个边界框的 IoU

    Args:
        box1: (xmin, ymin, xmax, ymax)
        box2: (xmin, ymin, xmax, ymax)

    Returns:
        iou: IoU 值，范围 [0, 1]
    """
    # 计算交集
    inter_xmin = max(box1[0], box2[0])
    inter_ymin = max(box1[1], box2[1])
    inter_xmax = min(box1[2], box2[2])
    inter_ymax = min(box1[3], box2[3])

    # 交集面积（如果框不重叠则为 0）
    inter_width = max(0, inter_xmax - inter_xmin)
    inter_height = max(0, inter_ymax - inter_ymin)
    inter_area = inter_width * inter_height

    # 各框面积
    area1 = (box1[2] - box1[0]) * (box1[3] - box1[1])
    area2 = (box2[2] - box2[0]) * (box2[3] - box2[1])

    # 并集面积
    union_area = area1 + area2 - inter_area

    # IoU
    iou = inter_area / union_area if union_area > 0 else 0

    return iou


def corner_to_center(box):
    """角点格式转中心点格式"""
    xmin, ymin, xmax, ymax = box
    x_center = (xmin + xmax) / 2
    y_center = (ymin + ymax) / 2
    width = xmax - xmin
    height = ymax - ymin
    return (x_center, y_center, width, height)


def center_to_corner(box):
    """中心点格式转角点格式"""
    x_center, y_center, width, height = box
    xmin = x_center - width / 2
    ymin = y_center - height / 2
    xmax = x_center + width / 2
    ymax = y_center + height / 2
    return (xmin, ymin, xmax, ymax)


# === 演示 ===
print("=== 目标检测基础演示 ===")

# 模拟边界框
ground_truth = (100, 100, 200, 200)  # 真实框
prediction1 = (110, 110, 190, 190)   # 较好的预测
prediction2 = (50, 50, 150, 150)     # 较差的预测
prediction3 = (300, 300, 400, 400)   # 完全偏离

print("\n真实框:", ground_truth)
print("预测框1:", prediction1)
print("预测框2:", prediction2)
print("预测框3:", prediction3)

print("\n=== IoU 计算 ===")
iou1 = calculate_iou(ground_truth, prediction1)
iou2 = calculate_iou(ground_truth, prediction2)
iou3 = calculate_iou(ground_truth, prediction3)

print(f"预测框1 IoU: {iou1:.3f} (较好)")
print(f"预测框2 IoU: {iou2:.3f} (部分重叠)")
print(f"预测框3 IoU: {iou3:.3f} (完全偏离)")

print("\n=== IoU 解读 ===")
print("IoU > 0.5: 一般认为检测成功")
print("IoU > 0.7: 检测质量较好")
print("IoU = 1.0: 完美匹配")

# === 格式转换演示 ===
print("\n=== 边界框格式转换 ===")
center_format = corner_to_center(ground_truth)
print(f"角点格式: {ground_truth}")
print(f"中心点格式: {center_format}")

back_to_corner = center_to_corner(center_format)
print(f"转换回去: {back_to_corner}")

print("\n=== 目标检测关键概念 ===")
print("1. 边界框: 用矩形表示物体位置")
print("2. IoU: 衡量两个框的重合程度")
print("3. 锚框: 预设的候选框模板")
print("4. 目标检测 = 分类 +定位")
```

## 避坑指南

1. **坐标超出图像范围**：预测框可能在图像边界外，需要裁剪。

2. **IoU 计算顺序错误**：交集要先取 max(左边界) 和 min(右边界)，顺序不能反。

3. **忽略面积为零的情况**：当框不重叠时，交集面积为 0，要正确处理。
