# 8.1 更深的网络

## 学习目标

- 理解为什么更深的网络能学到更多
- 了解 VGGNet、GoogLeNet、ResNet 等经典网络
- 理解残差学习（ResNet）的核心思想

## 核心概念

### 为什么要更深？

深度学习中的"深"指的是 **层数多**。

为什么层数多了就能学得更好？

**类比：** 你要描述一个人。
- 1 层：只能说"有五官"（低级特征）。
- 3 层：能说"大眼睛、高鼻子"（中级特征）。
- 10 层：能说"像个电影明星"（高级特征）。

**层数越多，能提取的特征就越高级、越抽象。**

## 原理解析

### 经典网络演进

| 网络 | 年份 | 层数 | 关键创新 |
|------|------|------|---------|
| AlexNet | 2012 | 8 层 | ReLU + Dropout + GPU 训练 |
| VGGNet | 2014 | 19 层 | 小卷积核 (3×3) 堆叠 |
| GoogLeNet | 2014 | 22 层 | Inception 模块（多尺度并行） |
| ResNet | 2015 | 152 层 | 残差连接（跳过几层） |

### ResNet 的残差学习

ResNet 解决了一个关键问题：**当网络变得非常深时，反而学不好了**（不是过拟合，而是训练数据上的准确率都下降了）。

**残差连接的思想：** 与其让网络直接学习想要的映射 H(x)，不如让网络学习"残差" F(x) = H(x) - x，然后输出 F(x) + x。

**大白话版：** 网络不需要从头学"这个输入应该变成什么"，只需要学"这个输入还需要做哪些微调"。

**类比：** 你有一篇已经写得不错的文章。与其重写，不如让编辑告诉你"这里改几个字、那里加一句"。修改比从头写容易得多。

## 代码实战

```python
import numpy as np

def relu(x):
    return np.maximum(0, x)

class ResidualBlock:
    """残差块：学习 F(x) = H(x) - x，输出 F(x) + x"""

    def __init__(self, input_size, hidden_size):
        self.W1 = np.random.randn(input_size, hidden_size) * np.sqrt(2.0 / input_size)
        self.b1 = np.zeros(hidden_size)
        self.W2 = np.random.randn(hidden_size, input_size) * np.sqrt(2.0 / hidden_size)
        self.b2 = np.zeros(input_size)

    def forward(self, x):
        """前向传播：x → 全连接 → ReLU → 全连接 → + x（残差连接）"""
        h = relu(np.dot(x, self.W1) + self.b1)
        out = np.dot(h, self.W2) + self.b2
        return out + x  # 残差连接：输出 = F(x) + x

class DeepNetwork:
    """用残差块堆叠的深层网络"""

    def __init__(self, input_size, hidden_size, num_blocks, output_size):
        self.blocks = []
        for _ in range(num_blocks):
            self.blocks.append(ResidualBlock(input_size, hidden_size))
        
        # 最后的分类层
        self.W_out = np.random.randn(input_size, output_size) * 0.01
        self.b_out = np.zeros(output_size)

    def predict(self, x):
        """前向传播"""
        for block in self.blocks:
            x = block.forward(x)
        return np.dot(x, self.W_out) + self.b_out

# 测试
if __name__ == "__main__":
    np.random.seed(42)
    
    # 创建一个有 5 个残差块的深层网络
    net = DeepNetwork(input_size=64, hidden_size=32, num_blocks=5, output_size=10)
    
    # 随机输入
    x = np.random.randn(1, 64)
    
    y = net.predict(x)
    print("输入形状:", x.shape)
    print("输出形状:", y.shape)
    print("输出值 (前5个):", y[0][:5])
    
    # 验证残差连接的效果：输入和输出的量级应该差不多
    print("\n输入幅度:", np.linalg.norm(x))
    print("输出幅度:", np.linalg.norm(y))
    print("残差连接让网络即使不学什么，输出也不会偏离输入太远")
```

**逐行解析：**

1. `import numpy as np` - 导入 NumPy 库，用于矩阵运算。
2. `def relu(x):` - 定义 ReLU 激活函数，负数变 0，正数不变。
3. `class ResidualBlock:` - 定义残差块类。
4. `self.W1 = np.random.randn(...) * np.sqrt(2.0 / input_size)` - 用 He 初始化（适合 ReLU），让初始权重不会太大也不会太小。
5. `h = relu(np.dot(x, self.W1) + self.b1)` - 第一层全连接 + ReLU。
6. `out = np.dot(h, self.W2) + self.b2` - 第二层全连接。
7. `return out + x` - **关键行**：残差连接，把输入 x 直接加到输出上。
8. `class DeepNetwork:` - 定义深层网络，用多个残差块堆叠。
9. `for block in self.blocks: x = block.forward(x)` - 依次通过每个残差块。
10. `return np.dot(x, self.W_out) + self.b_out` - 最后的分类层，输出最终结果。
11. `np.linalg.norm(x)` - 计算向量的幅度（长度），用来对比输入输出的量级。

## 避坑指南

1. **残差连接要求维度一致** - F(x) 和 x 的形状必须一样才能相加。如果不一样，需要用 1×1 卷积调整维度。
2. **不是越深越好** - 层数多了训练会变慢，也容易过拟合。需要根据数据量选择合适的深度。
3. **Batch Normalization 在深网络中几乎必用** - 没有 BN 的深层网络很难训练。
