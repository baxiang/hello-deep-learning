# 8.1 更深的网络

## 学习目标

- 理解为什么更深的网络能学到更多
- 了解 ResNet 的残差学习思想

## 核心概念

### 为什么要更深？

"深" = 层数多。

**类比：** 你要描述一个人。
- 1 层：只能说"有五官"
- 3 层：能说"大眼睛、高鼻子"
- 10 层：能说"像个电影明星"

**层数越多，能提取的特征越高级。**

## 原理解析

### ResNet 的残差学习

当网络变得非常深时，反而学不好了（不是过拟合，是训练数据上的准确率都下降了）。

**残差连接的思想：** 不直接学"输入应该变成什么"，而是学"输入还需要做哪些微调"。

```
输出 = F(x) + x
```

- `F(x)` 是网络学出来的"微调"
- `x` 是原始输入（直接跳过网络，叫"跳过连接"）

**类比：** 你有一篇写得不错的文章。与其重写，不如让编辑告诉你"这里改几个字"。修改比重写容易。

## 代码实战

```python
import numpy as np

def relu(x):
    return np.maximum(0, x)

class ResidualBlock:
    """残差块"""
    def __init__(self, input_size, hidden_size):
        np.random.seed(42)
        self.W1 = np.random.randn(input_size, hidden_size) * \
                  np.sqrt(2.0 / input_size)
        self.b1 = np.zeros(hidden_size)
        self.W2 = np.random.randn(hidden_size, input_size) * \
                  np.sqrt(2.0 / hidden_size)
        self.b2 = np.zeros(input_size)

    def forward(self, x):
        h = relu(np.dot(x, self.W1) + self.b1)
        out = np.dot(h, self.W2) + self.b2
        return out + x  # 残差连接：输出 = F(x) + x

# 测试
block = ResidualBlock(64, 32)
x = np.random.randn(1, 64)
y = block.forward(x)

print(f"输入形状: {x.shape}")
print(f"输出形状: {y.shape}")
print(f"输入幅度: {np.linalg.norm(x):.4f}")
print(f"输出幅度: {np.linalg.norm(y):.4f}")
print("→ 残差连接让输出不会偏离输入太远")
```
