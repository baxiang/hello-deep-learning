# 8.2 超参数调优

## 学习目标

- 理解什么是超参数
- 掌握超参数的搜索方法

## 核心概念

### 什么是超参数？

超参数是**训练前需要你手动设置的参数**，不是网络自己学出来的。

常见的超参数：
- 学习率
- 批次大小
- 网络层数
- 权重衰减系数
- Dropout 比例

**类比：** 超参数就像烤蛋糕的温度和时间。配方（网络结构）定了，但烤多久、多少度需要你试。

## 原理解析

### 搜索策略

| 方法 | 做法 | 优缺点 |
|------|------|--------|
| 随机搜索 | 在合理范围内随机选值 | 比网格搜索高效，推荐 |
| 网格搜索 | 所有组合都试一遍 | 参数多时非常慢 |
| 贝叶斯优化 | 根据之前结果智能选值 | 更高级但复杂 |

### 推荐范围

| 超参数 | 范围 | 说明 |
|--------|------|------|
| 学习率 | 10⁻⁶ ~ 10⁻² | 对数尺度搜索 |
| 批次大小 | 16 ~ 256 | 2 的幂次 |
| 权重衰减 | 10⁻⁸ ~ 10⁻⁴ | 通常很小 |
| Dropout | 0 ~ 0.5 | 0 表示不用 |

### 验证集

**不要用测试集调超参数！** 应该：
- 训练集 → 训练网络
- 验证集 → 调超参数
- 测试集 → 最终评估

## 代码实战

```python
import numpy as np

np.random.seed(42)
best_score = 0
best_params = {}

print("=== 超参数随机搜索 ===\n")

for trial in range(5):
    lr = 10 ** np.random.uniform(-6, -2)
    batch = int(2 ** np.random.uniform(4, 8))
    dropout = np.random.uniform(0, 0.5)

    # 模拟评估（实际应运行完整训练）
    score = 0.5 + np.random.normal(0, 0.05)
    if 0.001 <= lr <= 0.01: score += 0.1
    if 32 <= batch <= 128: score += 0.05
    score = min(score, 0.99)

    print(f"试验 {trial+1}: lr={lr:.6f}, "
          f"batch={batch}, dropout={dropout:.2f}, "
          f"得分={score:.4f}")

    if score > best_score:
        best_score = score
        best_params = {'lr': lr, 'batch': batch, 'dropout': dropout}

print(f"\n最佳: 得分 {best_score:.4f}")
print(f"参数: {best_params}")
```

## 避坑指南

1. **不用测试集调参** - 否则测试集失去独立评估的意义。
2. **记录每次结果** - 方便分析和复现。
