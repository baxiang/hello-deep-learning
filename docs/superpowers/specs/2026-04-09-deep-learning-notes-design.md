# 深度学习从零开始学习笔记 - 设计方案

## 1. 项目背景

基于《从零开始深度学习》(oreilly-japan/deep-learning-from-scratch) 一书，为 **0 基础的深度学习开发人员** 编写详细的学习笔记。

## 2. 受众定位

- **目标人群**：会写代码（了解基础编程逻辑），但从未接触过深度学习、神经网络、人工智能数学基础的开发者。
- **知识预设**：不需要预先掌握线性代数、微积分或概率论，所有涉及数学的地方都会在笔记中用最通俗的方式解释。
- **语言风格**：纯中文，不使用中英夹杂的术语（例如：不使用 "Loss Function"，而使用 "损失函数" 并辅以通俗解释）。

## 4. 单篇笔记结构模板

每篇 Markdown 笔记（例如 `01-什么是python.md`）遵循以下结构，确保学习者能够循序渐进地理解：

### 4.1 学习目标 (本节要学什么)
- 一句话概括本节内容。
- 列出 2-3 个具体的知识点（例如："学会使用 NumPy 创建二维数组"）。

### 4.2 核心概念 (大白话解释)
- **不使用公式**，用生活中的例子或比喻解释概念。
- 例如：解释“权重”时，比喻为“不同信号的重要性通道”。

### 4.3 原理解析 (深入理解)
- 在建立了直观理解后，再引入数学公式或代码逻辑。
- 配合简单的 ASCII 图示或文字描述图表来辅助理解数据流向。

### 4.4 代码实战 (从零手写)
- **禁止直接粘贴大段代码**。代码必须分段展示。
- 每一段代码后紧跟“代码解读”，解释每一行的作用，特别是 0 基础容易卡住的地方。
- 提供完整代码的存放路径（在 `code/` 目录下）。

### 4.5 避坑指南 (0 基础常见错误)
- 针对该知识点，初学者常犯的错误（例如：维度不匹配、除零错误等）。

### 4.6 课后思考
- 1-2 个引导性问题，鼓励读者修改代码并观察结果。

## 5. 各章内容规划（基于原书 8 章）

1.  **Python 基础**：环境搭建、NumPy 数组操作、Matplotlib 画图。重点是让读者熟悉工具。
2.  **感知机**：人工神经元的原始形态，与门、非门、异或门的实现。重点是“权重”和“偏置”的概念。
3.  **神经网络**：从感知机进化到神经网络，激活函数（Sigmoid, ReLU, Softmax）。重点是“非线性”的重要性。
4.  **学习算法**：损失函数、梯度下降法。重点是“机器是如何自动调整权重的”。
5.  **误差反向传播**：高效计算梯度的算法。重点是用“计算图”理解链式法则，而不是纯数学推导。
6.  **训练技巧**：SGD, Momentum, Adam, 权重初始化，Batch Normalization, Dropout。重点是“如何让网络学得更快更好”。
7.  **卷积神经网络 (CNN)**：解决图像识别问题。重点是卷积层如何处理空间结构信息。
8.  **深度学习进阶**：实际训练 MNIST，模型评估，超参数调整。重点是“完整的深度学习项目流程”。

## 6. 实施策略

- **分章实施**：每完成一章的笔记，作为一个可独立学习的单元。
- **代码验证**：所有笔记中的代码示例必须经过实际运行验证。
- **持续迭代**：根据学习者的反馈（假设有反馈）调整解释的深度。

## 7. 验收标准

1.  目录结构完全按照设计方案建立。
2.  每个 Markdown 文件都符合“单篇笔记结构模板”。
3.  语言纯中文，通俗易懂，无未经解释的专业术语。
4.  所有代码文件在 `code/` 目录下均可运行（假设环境配置正确）。
