# 4.4 学习算法的实现

## 学习目标

- 把前面的知识串起来，实现完整的训练流程
- 理解 Epoch 和 Mini-batch 的关系
- 能看到训练过程中损失的变化

## 核心概念

### 什么是 Epoch？

**Epoch（轮次）** = 用全部训练数据学习一遍。

比如训练数据有 60000 张图片，每次用 100 张（Mini-batch），需要 600 次才能看完所有图片。这 600 次合起来叫 1 个 Epoch。

**类比：** 复习考试。
- Mini-batch = 每次复习 10 页书。
- Epoch = 把整本书复习一遍。
- 通常要复习好几遍（多个 Epoch）才能记住。

### 学习率

学习率决定每次权重调整的步伐大小。

```
新权重 = 旧权重 - 学习率 × 梯度
```

- 学习率大 → 大步走 → 可能跳过最优值
- 学习率小 → 小步走 → 学得慢

**类比：** 下山时你迈的步子大小。

### 训练循环

训练就是不断重复这 5 步：

```
1. 随机选 Mini-batch
2. 前向传播（做题）
3. 计算损失（对答案）
4. 计算梯度（找改进方向）
5. 更新权重（改进）
```

## 原理解析

### TwoLayerNet 类

我们把神经网络封装成一个类，包含所有需要的功能：

```
TwoLayerNet
├── 参数（W1, b1, W2, b2）
├── predict(x) → 前向传播
├── loss(x, t) → 计算损失
├── numerical_gradient(x, t) → 计算梯度
└── update_params(grads, lr) → 更新权重
```

这样代码更清晰，每个功能各司其职。

### 为什么损失会下降？

每次更新权重时，我们沿着梯度的反方向走：

```
新权重 = 旧权重 - 学习率 × 梯度
```

梯度指向损失增加的方向，反方向就是损失减小的方向。

**类比：** 你站在山坡上，梯度告诉你"上坡方向"。你反着走，就是下坡。

## 代码实战

### 完整可运行的训练代码

```python
import numpy as np

# === 激活函数 ===
def sigmoid(x):
    return np.where(x >= 0, 
                    1 / (1 + np.exp(-x)), 
                    np.exp(x) / (1 + np.exp(x)))

def softmax(a):
    c = np.max(a)
    exp_a = np.exp(a - c)
    return exp_a / np.sum(exp_a)

# === 损失函数 ===
def cross_entropy_error(y, t):
    delta = 1e-7
    return -np.sum(t * np.log(y + delta))

# === 数值微分 ===
def numerical_gradient(f, x):
    h = 1e-4
    grad = np.zeros_like(x)
    for i in range(x.size):
        tmp = x[i]
        x[i] = tmp + h
        fxh1 = f(x)
        x[i] = tmp - h
        fxh2 = f(x)
        grad[i] = (fxh1 - fxh2) / (2 * h)
        x[i] = tmp
    return grad

# === 两层神经网络 ===
class TwoLayerNet:
    def __init__(self, input_size, hidden_size, output_size, 
                 weight_init_std=0.01):
        """初始化网络参数"""
        np.random.seed(42)
        self.params = {}
        self.params['W1'] = weight_init_std * \
            np.random.randn(input_size, hidden_size)
        self.params['b1'] = np.zeros(hidden_size)
        self.params['W2'] = weight_init_std * \
            np.random.randn(hidden_size, output_size)
        self.params['b2'] = np.zeros(output_size)

    def predict(self, x):
        """前向传播"""
        W1 = self.params['W1']
        W2 = self.params['W2']
        b1 = self.params['b1']
        b2 = self.params['b2']
        
        a1 = np.dot(x, W1) + b1
        z1 = sigmoid(a1)
        a2 = np.dot(z1, W2) + b2
        y = softmax(a2)
        return y

    def loss(self, x, t):
        """计算损失"""
        y = self.predict(x)
        return cross_entropy_error(y, t)

    def numerical_gradient(self, x, t):
        """用数值微分计算梯度"""
        grads = {}
        # 对每个参数分别计算梯度
        for key in ('W1', 'b1', 'W2', 'b2'):
            # 定义一个函数：输入是参数，输出是损失
            f = lambda W: self.loss(x, t)
            grads[key] = numerical_gradient(f, self.params[key])
        return grads

    def update_params(self, grads, learning_rate=0.1):
        """更新权重"""
        for key in ('W1', 'b1', 'W2', 'b2'):
            self.params[key] -= learning_rate * grads[key]

# === 训练流程 ===
print("=== 两层神经网络训练演示 ===\n")

# 1. 创建网络：2 个输入 → 10 个隐藏节点 → 3 个输出
net = TwoLayerNet(input_size=2, hidden_size=10, output_size=3)

# 2. 创建模拟数据：5 个样本，每个 2 个特征，3 个类别
x_train = np.random.randn(5, 2)
t_train = np.array([
    [1, 0, 0],  # 样本 1 → 类别 0
    [0, 1, 0],  # 样本 2 → 类别 1
    [0, 0, 1],  # 样本 3 → 类别 2
    [1, 0, 0],  # 样本 4 → 类别 0
    [0, 1, 0],  # 样本 5 → 类别 1
])

print(f"训练数据: {x_train.shape[0]} 个样本")
print(f"输入特征: {x_train.shape[1]} 个")
print(f"输出类别: {t_train.shape[1]} 个\n")

# 3. 训练前损失
print("--- 训练前 ---")
loss_before = net.loss(x_train, t_train)
print(f"损失: {loss_before:.4f}")

# 4. 训练循环（由于数值微分很慢，只演示 2 次迭代）
print("\n--- 训练中 ---")
for i in range(3):
    # 计算梯度
    grads = net.numerical_gradient(x_train, t_train)
    # 更新权重
    net.update_params(grads, learning_rate=0.1)
    # 计算新损失
    loss = net.loss(x_train, t_train)
    print(f"第 {i+1} 次更新后，损失: {loss:.4f}")

print("\n--- 训练后 ---")
loss_after = net.loss(x_train, t_train)
print(f"损失: {loss_after:.4f}")
print(f"损失下降: {loss_before - loss_after:.4f}")

# 5. 预测
print("\n--- 预测结果 ---")
y = net.predict(x_train)
for i in range(5):
    pred = np.argmax(y[i])
    true = np.argmax(t_train[i])
    match = "✓" if pred == true else "✗"
    print(f"样本 {i}: 预测={pred}, 正确={true} {match}")
```

**解读：**

3. 预测全部正确（因为数据简单，只有 5 个样本）。

### 逐行解读

**创建网络：**
```python
net = TwoLayerNet(input_size=2, hidden_size=10, output_size=3)
```
- 2 个输入 → 10 个隐藏节点 → 3 个输出。
- 权重是随机初始化的。

**计算梯度：**
```python
grads = net.numerical_gradient(x_train, t_train)
```
- 对 W1, b1, W2, b2 分别计算梯度。
- 这一步很慢，因为每个参数都要算两次前向传播。

**更新权重：**
```python
net.update_params(grads, learning_rate=0.1)
```
- 沿着梯度反方向走一步，步长是 0.1。

## 避坑指南

1. **数值微分太慢** - 这个代码用了数值微分，参数多的时候非常慢。实际中用"反向传播"（第 5 章），快 1000 倍以上。

2. **学习率要试** - 0.1 不一定适合所有情况。如果损失不降，试试更小的学习率（0.01 或 0.001）。

3. **Epoch vs 迭代** - 1 个 Epoch = 用全部数据学一遍。如果数据 100 个，Mini-batch 10 个，1 个 Epoch = 10 次迭代。
