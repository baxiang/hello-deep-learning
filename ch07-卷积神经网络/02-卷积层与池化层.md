# 7.2 卷积层与池化层

## 学习目标

- 理解卷积运算的过程
- 理解填充（Padding）和步幅（Stride）的作用
- 理解池化层的作用

## 核心概念

### 卷积运算（Convolution）

卷积运算就像用一个 **小窗口（滤波器）** 在图片上滑动，每次计算窗口内像素和滤波器的加权和。

**类比：** 你用手电筒照一张大纸。手电筒的光圈就是滤波器，你移动手电筒，每次照亮一小块区域。

### 池化运算（Pooling）

池化就是 **缩小图片**。在图片上滑动一个窗口，取窗口内的最大值（最大池化）或平均值（平均池化）。

**类比：** 你站在高处看城市，远处的建筑看不清楚细节，但大致轮廓还在。池化就是"站远一点看"。

## 原理解析

### 卷积运算详解

假设有一个 4×4 的输入数据和一个 3×3 的滤波器：

```
输入数据:          滤波器:
[1, 2, 3, 0]       [1, 0, -1]
[4, 5, 6, 1]       [0, 1,  0]
[7, 8, 9, 2]       [-1, 0, 1]
[0, 1, 2, 3]

第一步：滤波器覆盖左上角 3×3 区域
[1, 2, 3]
[4, 5, 6]   ·  滤波器  =  1×1 + 2×0 + 3×(-1) + 4×0 + 5×1 + 6×0 + 7×(-1) + 8×0 + 9×1 = 6
[7, 8, 9]
```

输出数据的大小 = (输入大小 - 滤波器大小 + 1) × (输入大小 - 滤波器大小 + 1)

### 填充（Padding）

填充就是在输入数据的周围补一圈 0。

**为什么要填充？** 如果不填充，每次卷积后图片会变小。填充可以保持图片大小不变。

### 步幅（Stride）

步幅就是滤波器每次滑多远。步幅为 1 就是每次滑一格，步幅为 2 就是每次滑两格。

**输出大小公式：**
```
输出大小 = (输入大小 + 2×填充 - 滤波器大小) / 步幅 + 1
```

### 三维数据

彩色图片是三维的（高 × 宽 × 通道），滤波器也是三维的。卷积运算是把三维的滤波器和三维的输入对应位置相乘后求和。

## 代码实战

```python
import numpy as np

def conv2d_simple(x, w, stride=1, pad=0):
    """简单的二维卷积（仅用于理解，不考虑偏置和批次）"""
    # x: 输入 (H, W), w: 滤波器 (FH, FW)
    H, W = x.shape
    FH, FW = w.shape
    out_h = (H + 2 * pad - FH) // stride + 1
    out_w = (W + 2 * pad - FW) // stride + 1
    
    # 填充
    if pad > 0:
        x_padded = np.pad(x, pad, mode='constant', constant_values=0)
    else:
        x_padded = x
    
    out = np.zeros((out_h, out_w))
    for y in range(out_h):
        for x_pos in range(out_w):
            y_start = y * stride
            x_start = x_pos * stride
            out[y, x_pos] = np.sum(x_padded[y_start:y_start+FH, x_start:x_start+FW] * w)
    
    return out

def max_pooling(x, pool_size=2, stride=2):
    """简单的最大池化"""
    H, W = x.shape
    out_h = (H - pool_size) // stride + 1
    out_w = (W - pool_size) // stride + 1
    
    out = np.zeros((out_h, out_w))
    for y in range(out_h):
        for x_pos in range(out_w):
            y_start = y * stride
            x_start = x_pos * stride
            out[y, x_pos] = np.max(x[y_start:y_start+pool_size, x_start:x_start+pool_size])
    
    return out

# 测试
if __name__ == "__main__":
    x = np.array([
        [1, 2, 3, 0],
        [4, 5, 6, 1],
        [7, 8, 9, 2],
        [0, 1, 2, 3]
    ], dtype=np.float64)
    
    w = np.array([
        [1, 0, -1],
        [0, 1, 0],
        [-1, 0, 1]
    ], dtype=np.float64)
    
    print("输入数据:")
    print(x)
    print("\n滤波器:")
    print(w)
    
    out = conv2d_simple(x, w)
    print("\n卷积输出:")
    print(out)
    
    pooled = max_pooling(x)
    print("\n最大池化输出:")
    print(pooled)
```

## 避坑指南

1. **输出大小算错** - 用公式 `(输入 + 2×填充 - 滤波器) / 步幅 + 1` 验证。如果除不尽，说明参数设错了。
2. **池化层没有参数** - 池化层不需要学习参数，它只是取最大值或平均值。这也是为什么池化层不容易过拟合。
3. **通道数在卷积后不变** - 输出数据的通道数 = 滤波器的个数。如果有 32 个滤波器，输出就有 32 个通道。
