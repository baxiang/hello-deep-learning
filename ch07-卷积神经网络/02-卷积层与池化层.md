# 7.2 卷积层与池化层

## 学习目标

- 理解卷积运算的过程
- 理解填充（Padding）和步幅（Stride）
- 理解池化层的作用

## 核心概念

### 卷积运算：用"小窗口"看图片

卷积运算就像用一个**小窗口（滤波器）**在图片上滑动，每次计算窗口内像素和滤波器的加权和。

**类比：** 你用手电筒照一张大纸。手电筒的光圈就是滤波器，你移动手电筒，每次照亮一小块区域。

### 池化运算：缩小图片

池化就是在图片上滑动一个窗口，取窗口内的最大值（最大池化）或平均值（平均池化）。

**类比：** 你站在高处看城市，远处的建筑看不清楚细节，但大致轮廓还在。池化就是"站远一点看"。

## 原理解析

### 卷积运算详解

假设有一个 4×4 的输入和一个 3×3 的滤波器：

```
输入:             滤波器:
[1, 2, 3, 0]      [1,  0, -1]
[4, 5, 6, 1]      [0,  1,  0]
[7, 8, 9, 2]      [-1, 0,  1]
[0, 1, 2, 3]

第一步：滤波器覆盖左上角 3×3
[1, 2, 3]
[4, 5, 6]  ·  滤波器  =  1×1 + 2×0 + 3×(-1) + ... + 9×1 = 6
[7, 8, 9]
```

**输出大小公式：**
```
输出 = (输入 - 滤波器 + 1) × (输入 - 滤波器 + 1)
     = (4 - 3 + 1) × (4 - 3 + 1) = 2 × 2
```

### 填充（Padding）

在图片周围补一圈 0。

**为什么？** 不填充的话，每次卷积图片会变小。填充可以保持大小不变。

### 步幅（Stride）

滤波器每次滑多远。
- 步幅 1 = 每次滑 1 格
- 步幅 2 = 每次滑 2 格（输出会变小）

### 池化

最大池化：取窗口内的最大值。

```
输入 4×4:        池化窗口 2×2, 步幅 2:
[1, 2, 3, 0]     max(1,2,4,5)=5  max(3,0,6,1)=6
[4, 5, 6, 1]  →  [5,             6]
[7, 8, 9, 2]     [9,             9]
[0, 1, 2, 3]
```

## 代码实战

### 完整可运行代码

```python
import numpy as np

def simple_conv2d(x, w, stride=1, pad=0):
    """简单的二维卷积"""
    H, W = x.shape
    FH, FW = w.shape
    out_h = (H + 2*pad - FH) // stride + 1
    out_w = (W + 2*pad - FW) // stride + 1

    if pad > 0:
        x = np.pad(x, pad, mode='constant', constant_values=0)

    out = np.zeros((out_h, out_w))
    for y in range(out_h):
        for xi in range(out_w):
            ys = y * stride
            xs = xi * stride
            out[y, xi] = np.sum(x[ys:ys+FH, xs:xs+FW] * w)
    return out

def simple_max_pooling(x, pool_size=2, stride=2):
    """简单的最大池化"""
    H, W = x.shape
    out_h = (H - pool_size) // stride + 1
    out_w = (W - pool_size) // stride + 1

    out = np.zeros((out_h, out_w))
    for y in range(out_h):
        for xi in range(out_w):
            ys = y * stride
            xs = xi * stride
            out[y, xi] = np.max(x[ys:ys+pool_size, xs:xs+pool_size])
    return out

# === 测试 ===
print("=== 卷积和池化测试 ===\n")

x = np.array([
    [1, 2, 3, 0],
    [4, 5, 6, 1],
    [7, 8, 9, 2],
    [0, 1, 2, 3]
], dtype=np.float64)

w = np.array([
    [1, 0, -1],
    [0, 1, 0],
    [-1, 0, 1]
], dtype=np.float64)

print("输入 (4×4):")
print(x)
print("\n滤波器 (3×3):")
print(w)

out = simple_conv2d(x, w)
print(f"\n卷积输出 ({out.shape[0]}×{out.shape[1]}):")
print(out)

pooled = simple_max_pooling(x)
print(f"\n最大池化输出 ({pooled.shape[0]}×{pooled.shape[1]}):")
print(pooled)
```

### 运行结果

## 避坑指南

1. **输出大小算错** - 用公式 `(输入 + 2×填充 - 滤波器) / 步幅 + 1` 验证。

2. **池化层没有参数** - 池化不需要学习参数，只是取最大值或平均值。

3. **通道数 = 滤波器个数** - 如果有 32 个滤波器，输出就有 32 个通道。
