# 12.3 情感分析

## 学习目标

- 理解情感分析的任务定义
- 掌握文本分类的基本方法
- 了解RNN 和CNN 在NLP 的应用

## 核心概念

### 情感分析：读懂情绪

**情感分析：判断文本的情感倾向。**

| 输入 | 输出 |
|------|------|
| "这部电影太精彩了！" | 正面（褒义） |
| "服务太差了，很失望。" | 负面（贬义） |
| "今天天气一般。" | 中性 |

情感分析是**文本分类**的一种：
- 输入：一段文本
- 输出：类别标签（褒/贬/中性）

### 应用场景

| 场景 | 作用 |
|------|------|
| 电商评论分析 | 了解用户对产品的满意度 |
| 社交媒体监控 | 分析公众对事件的情绪倾向 |
| 客户服务 | 自动识别投诉并优先处理 |
| 舆情分析 | 监控品牌或人物的网络口碑 |

### 情感分析的难点

1. **隐含情感**：
   ```
   "这手机电池能用一天" → 正面还是负面？
   （取决于用户期望，可能需要上下文）
   ```

2. **反讽/讽刺**：
   ```
   "这服务真是'好'极了" → 表面褒义，实际贬义
   ```

3. **混合情感**：
   ```
   "菜品不错，但服务太慢" → 同时有褒贬
   ```

## 原理解析

### 文本分类的流程

```
原始文本 → 预处理 → 词向量 → 模型 → 分类结果
```

**预处理：**
- 分词（中文需要）
- 去停用词（可选）
- 标准化（去除标点、大小写统一）

**词向量：**
- 把词变成向量
- 可以用预训练词向量（Word2Vec）
- 或让模型自己学习

**模型：**
- RNN：按顺序处理，理解上下文
- CNN：提取局部特征（如短语）
- Transformer：全局注意力

### RNN 方法

用 RNN（或 LSTM/GRU）处理文本序列：

```
词向量序列 → LSTM → 最后隐状态 → 分类层 → 情感标签
```

LSTM 能理解：
- "这部电影虽然开头无聊，但后面越来越精彩"
- LSTM 记住"虽然"后面的转折，理解整体是褒义

### CNN 方法

用 CNN 提取文本的局部特征：

```
词向量序列 → 1D CNN → 最大池化 → 分类层 → 情感标签
```

CNN 能提取：
- 短语模式（如"太精彩了"、"很失望"）
- 这些短语直接表达情感

### 结合预训练模型

用 BERT 等预训练模型：

```
文本 → BERT → [CLS] 向量 → 分类层 → 情感标签
```

BERT 已经学到了丰富的语言知识，微调后效果通常最好。

## 代码实战

### 情感分析流程演示

```python
import numpy as np


def preprocess(text):
    """简单的文本预处理"""
    # 分词（简化：假设已经分词）
    words = text.split()
    # 去除标点
    words = [w.strip("，。！？") for w in words]
    return words


def words_to_vectors(words, word_to_vec, max_len=10):
    """词向量转换"""
    vectors = []
    for word in words[:max_len]:
        if word in word_to_vec:
            vectors.append(word_to_vec[word])
        else:
            vectors.append(np.zeros_like(word_to_vec["我"]))  # 未知词

    # 填充到固定长度
    while len(vectors) < max_len:
        vectors.append(np.zeros_like(word_to_vec["我"]))

    return np.array(vectors)


def simple_rnn_forward(x_seq, hidden_size=8):
    """简化的 RNN 前向传播"""
    h = np.zeros(hidden_size)

    for x in x_seq:
        # 简化的 RNN 更新
        h = np.tanh(x[:hidden_size] + h)

    return h


def classify(features, num_classes=3):
    """分类层"""
    # 简化：用特征的均值作为分类依据
    np.random.seed(42)
    W = np.random.randn(len(features), num_classes) * 0.1
    logits = np.dot(features, W)
    probs = np.exp(logits) / np.sum(np.exp(logits))
    return probs


# === 演示 ===
np.random.seed(42)

print("=== 情感分析流程演示 ===")

# 模拟词向量
word_to_vec = {
    "这": np.random.randn(16),
    "电影": np.random.randn(16),
    "太": np.random.randn(16),
    "精彩": np.random.randn(16),
    "了": np.random.randn(16),
    "很": np.random.randn(16),
    "失望": np.random.randn(16),
    "不错": np.random.randn(16),
    "无聊": np.random.randn(16),
    "我": np.random.randn(16),
}

# 测试句子
texts = [
    "这电影太精彩了",
    "很失望",
    "还不错",
]

labels = ["褒义", "贬义", "中性"]

print("\n=== 文本处理流程 ===")
for text in texts:
    print(f"\n输入文本: '{text}'")

    # 预处理
    words = preprocess(text)
    print(f"分词: {words}")

    # 词向量
    vectors = words_to_vectors(words, word_to_vec)
    print(f"词向量形状: {vectors.shape}")

    # RNN提取特征
    features = simple_rnn_forward(vectors)
    print(f"RNN 输出形状: {features.shape}")

    # 分类
    probs = classify(features)
    predicted_label = labels[np.argmax(probs)]
    print(f"分类概率: 褒={probs[0]:.2f}, 贬={probs[1]:.2f}, 中={probs[2]:.2f}")
    print(f"预测: {predicted_label}")

print("\n=== 情感分析关键步骤 ===")
print("1. 预处理：分词、清洗")
print("2. 词向量：词变数字")
print("3. 模型：RNN/CNN/Transformer 提取特征")
print("4. 分类：输出情感类别")
```

## 避坑指南

1. **中文分词质量**：分词错误会影响效果，如"深度学习"不应分成"深度"、"学习"。

2. **忽略上下文**：单独看"精彩"是褒义，但"不精彩"是贬义，模型需要理解上下文。

3. **数据不平衡**：实际数据中褒贬比例可能不均衡，需要调整训练策略。
