## 核心概念

### 文本分类：给文本贴标签

**文本分类是最基础的 NLP 任务。**

| 任务 | 输入 | 输出类别 |
|------|------|----------|
| 情感分析 | 评论 | 褒/贬/中性 |
| 新闻分类 | 新闻内容 | 体育/财经/科技 |
| 垃圾邮件检测 | 邮件内容 | 正常/垃圾 |
| 话题分类 | 社交帖子 | 政治/娱乐/生活 |

### 文本分类的流程

```
原始文本 → 预处理 → 向量化 → 模型 → 预测
```

### 文本预处理

| 步骤 | 作用 | 方法 |
|------|------|------|
| 分词 | 把句子拆成词 | 中文：jieba等工具 |
| 去停用词 | 删除无意义词 | 删除"的"、"是"等 |
| 标准化 | 统一格式 | 小写、去标点 |
| 填充/截断 | 固定长度 | 方便批量处理 |

### 向量化方法

| 方法 | 说明 |
|------|------|
| One-hot | 每个词一个稀疏向量 |
| 词向量 | Word2Vec/GloVe 等预训练 |
| TF-IDF | 词的重要性加权 |
| 模型学习 | 让模型自己学习词表示 |

## 原理解析

### 分词

**中文分词**：中文没有天然空格分隔，需要专门处理。

```
原句："我喜欢深度学习"
分词：["我", "喜欢", "深度", "学习"]
```

分词工具：jieba、pkuseg、HanLP 等

**英文分词**：直接按空格分，但需要处理特殊情况。

```
原句："Don't stop learning"
分词：["Don", "'t", "stop", "learning"]  或["Do", "n't", ...]
```

### 填充和截断

模型需要固定长度的输入：

```
短文本 → 填充到固定长度
长文本 → 截断到固定长度
```

例如固定长度 10：
```
["我", "喜欢", "深度学习"] → ["我", "喜欢", "深度", "学习", [PAD], [PAD], ...]
```

### 批处理

训练时需要一次处理多条文本：

```
文本1: 长度 5
文本2: 长度 8
文本3: 长度 3
→ 全部填充到长度 8
→ 形成批次 (batch_size, max_len, vector_dim)
```

### 词表和索引

建立词表（词汇表）：
```
词表 = {"我": 0, "喜欢": 1, "深度": 2, "学习": 3, ...}
[PAD]: 填充标记
[UNK]: 未知词标记
```

文本转索引：
```
"我喜欢学习" → [0, 1, 3]
```

## 代码实战

### 文本预处理工具

```python
import numpy as np


class TextProcessor:
    """文本预处理工具"""

    def __init__(self, vocab=None, max_len=20):
        """初始化

        Args:
            vocab: 词表（词到索引的映射）
            max_len: 最大序列长度
        """
        self.vocab = vocab or {}
        self.max_len = max_len

        # 特殊标记
        self.pad_idx = 0
        self.unk_idx = 1

    def tokenize(self, text):
        """分词（简化版）"""
        # 中文：假设已经分词，按空格分
        # 实际应用用jieba 等工具
        words = text.split()
        return words

    def words_to_indices(self, words):
        """词转索引"""
        indices = []
        for word in words:
            if word in self.vocab:
                indices.append(self.vocab[word])
            else:
                indices.append(self.unk_idx)  # 未知词
        return indices

    def pad_sequence(self, indices):
        """填充到固定长度"""
        if len(indices) < self.max_len:
            # 填充
            indices = indices + [self.pad_idx] * (self.max_len - len(indices))
        else:
            # 截断
            indices = indices[:self.max_len]
        return indices

    def process(self, text):
        """完整预处理流程"""
        words = self.tokenize(text)
        indices = self.words_to_indices(words)
        indices = self.pad_sequence(indices)
        return np.array(indices)

    def build_vocab(self, texts, min_freq=1):
        """从文本构建词表"""
        word_counts = {}
        for text in texts:
            words = self.tokenize(text)
            for word in words:
                word_counts[word] = word_counts.get(word, 0) + 1

        # 过滤低频词
        self.vocab = {"[PAD]": 0, "[UNK]": 1}
        idx = 2
        for word, count in word_counts.items():
            if count >= min_freq:
                self.vocab[word] = idx
                idx += 1

        return self.vocab


def create_batches(texts, labels, batch_size, processor):
    """创建批次数据"""
    batches = []

    for i in range(0, len(texts), batch_size):
        batch_texts = texts[i:i + batch_size]
        batch_labels = labels[i:i + batch_size]

        # 处理文本
        batch_indices = [processor.process(t) for t in batch_texts]
        batch_indices = np.array(batch_indices)

        batches.append((batch_indices, np.array(batch_labels)))

    return batches


# === 演示 ===
print("=== 文本分类预处理演示 ===")

# 示例数据
texts = [
    "我 喜欢 深度 学习",
    "这 电影 很 精彩",
    "服务 太 差 了",
]
labels = [1, 1, 0]  # 1=褒义, 0=贬义

# 构建词表
processor = TextProcessor(max_len=10)
vocab = processor.build_vocab(texts)

print(f"\n词表大小: {len(vocab)}")
print(f"词表内容: {vocab}")

# 处理单条文本
print("\n=== 单条文本处理 ===")
text = "我 喜欢 学习"
print(f"原文: '{text}'")

words = processor.tokenize(text)
print(f"分词: {words}")

indices = processor.words_to_indices(words)
print(f"索引: {indices}")

padded = processor.pad_sequence(indices)
print(f"填充后: {padded}")

# 完整流程
processed = processor.process(text)
print(f"完整处理结果: {processed}")

# 批处理
print("\n=== 批处理 ===")
batches = create_batches(texts, labels, batch_size=2, processor=processor)

for i, (batch_x, batch_y) in enumerate(batches):
    print(f"批次{i}:")
    print(f"  输入形状: {batch_x.shape}")
    print(f"  输入内容: {batch_x}")
    print(f"  标签: {batch_y}")

print("\n=== 文本分类关键步骤 ===")
print("1. 分词：把句子拆成词")
print("2. 构建词表：词到索引的映射")
print("3. 向量化：词索引转向量")
print("4. 填充/截断：统一长度")
print("5. 批处理：方便训练")
print("6. 模型训练：分类学习")
```

## 避坑指南

1. **词表太小**：很多词变成[UNK]，模型无法理解。

2. **词表太大**：计算开销大，低频词噪声多。

3. **填充位置**：某些模型需要忽略填充位置的损失计算。

4. **截断策略**：截断开头还是结尾？可能影响效果。
