## 核心概念

### 预训练：先学通用能力，再学专业技能

**传统方法：每个任务从头训练。**

情感分类 → 收集标注数据 → 从头训练模型
文本分类 → 收集标注数据 → 从头训练模型
问答系统 → 收集标注数据 → 从头训练模型

问题：
- 需要大量标注数据
- 模型只学到一个任务的知识

**预训练方法：先在大数据上学通用能力。**

预训练阶段：
- 用海量无标注文本（如整个互联网）
- 学习语言的通用规律（语法、语义、常识）

微调阶段：
- 用少量标注数据
- 调整模型适应具体任务

**类比：教育体系。**

传统 → 每个职业单独培训（医生、律师分别从头学）
预训练 → 先上小学中学（学通用知识），再上大学专修

### BERT：双向理解

BERT（Bidirectional Encoder Representations from Transformers）是预训练语言模型的代表。

**双向：同时看左边和右边的上下文。**

传统模型（如 GPT）：
- 只能从左到右看
- 理解"我喜欢"时，还没看到后面的词

BERT：
- 同时看左右两边
- 理解"我喜欢"时，能看到整句话

**这就像读句子时，可以来回看，理解更准确。**

### BERT 的两个预训练任务

| 任务 | 全称 | 目的 |
|------|------|------|
| MLM | Masked Language Model | 学习理解词汇和上下文 |
| NSP | Next Sentence Prediction | 学习理解句子关系 |

**MLM：把词盖住，让模型猜。**

```
原句："我喜欢深度学习"
遮盖："我喜欢[MASK]学习"
目标：猜[MASK]是什么（"深度"）
```

**NSP：判断两句话是否连续。**

```
句子A："我喜欢深度学习"
句子B："它很有趣"
目标：判断B是否是A的下一句（是）
```

## 原理解析

### MLM 的设计

随机遮盖 15% 的词：
- 80% 替换成 [MASK]
- 10% 替换成随机词
- 10% 保持不变

为什么这样设计？
- 如果全是 [MASK]，模型只学"填空"
- 加入随机词和原词，让模型更鲁棒

### NSP 的设计

构造训练数据：
- 50% 是真正的下一句（正例）
- 50% 是随机句子（负例）

模型输出：二分类（是否连续）

这让BERT 学到句子间的逻辑关系。

### BERT 的结构

BERT 基于 Transformer Encoder：
- 多层自注意力
- 双向理解上下文
- 输出每个位置的表示

BERT 的输出：
- 每个词的向量表示
- 整句话的向量表示（[CLS] 位置）

### 微调 BERT

预训练后的 BERT 可以用于多种任务：

| 任务 | 微调方式 |
|------|----------|
| 情感分类 | 用[CLS] 输出 + 分类层 |
| 文本分类 | 用[CLS] 输出 + 分类层 |
| 问答系统 | 用两个输出预测答案位置 |
|命名实体识别 | 每个词输出 + 分类层 |

**[CLS] 是 BERT 的特殊标记，放在句首，其输出用于句子级别的任务。**

## 代码实战

### BERT 概念演示

```python
import numpy as np


def simulate_mlm(sentence, mask_prob=0.15):
    """模拟 MLM 任务

    Args:
        sentence: 原句子（词列表）
        mask_prob: 遮盖概率

    Returns:
        masked_sentence: 遮盖后的句子
        masked_positions: 被遮盖的位置
        original_tokens: 原来的词
    """
    masked_sentence = sentence.copy()
    masked_positions = []
    original_tokens = []

    for i in range(len(sentence)):
        if np.random.random() < mask_prob:
            masked_positions.append(i)
            original_tokens.append(sentence[i])

            # 80% MASK, 10% 随机词, 10% 保持
            r = np.random.random()
            if r < 0.8:
                masked_sentence[i] = "[MASK]"
            elif r < 0.9:
                masked_sentence[i] = np.random.choice(["我", "你", "他", "她", "它"])
            # else: 保持不变

    return masked_sentence, masked_positions, original_tokens


def simulate_nsp(sentence_a, sentence_b, is_next=True):
    """模拟 NSP 任务

    Args:
        sentence_a: 第一句
        sentence_b: 第二句
        is_next: 是否真的是下一句

    Returns:
        输入格式和标签
    """
    input_text = ["[CLS]"] + sentence_a + ["[SEP]"] + sentence_b + ["[SEP]"]
    label = 1 if is_next else 0

    return input_text, label


# === 演示 ===
np.random.seed(42)

print("=== BERT 预训练任务演示 ===")

# MLM 演示
sentence = ["我", "喜欢", "深度", "学习", "很", "有趣"]
masked, positions, originals = simulate_mlm(sentence)

print("\n=== MLM 任务 ===")
print(f"原句子: {sentence}")
print(f"遮盖后: {masked}")
print(f"遮盖位置: {positions}")
print(f"原词: {originals}")
print("任务: 模型需要预测[MASK]位置原来的词")

# NSP 演示
print("\n=== NSP 任务 ===")

sentence_a = ["我", "喜欢", "深度", "学习"]
sentence_b = ["它", "很", "有趣"]  # 真正的下一句
sentence_c = ["今天", "天气", "不错"]  # 随机句子

input1, label1 = simulate_nsp(sentence_a, sentence_b, is_next=True)
input2, label2 = simulate_nsp(sentence_a, sentence_c, is_next=False)

print(f"正例输入: {input1}")
print(f"正例标签: {label1} (是下一句)")

print(f"负例输入: {input2}")
print(f"负例标签: {label2} (不是下一句)")

print("\n=== BERT 关键设计 ===")
print("1. 双向理解：同时看左右上下文")
print("2. MLM：遮盖词预测，学习词汇理解")
print("3. NSP：句子关系预测，学习句子理解")
print("4. 预训练 → 微调：先学通用，再学专业")
print("5. [CLS] 标记：句子级别的表示")
```

## 避坑指南

1. **[MASK] 只在预训练用**：实际应用时输入没有 [MASK]，模型要学会处理真实文本。

2. **预训练成本高**：BERT 预训练需要大量计算资源，一般直接用现成的预训练模型。

3. **微调数据量**：虽然比从头训练需要少，但太少也可能效果不好。
