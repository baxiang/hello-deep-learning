# 9.2 循环神经网络原理

## 学习目标

- 理解 RNN 的核心思想
- 掌握隐状态的概念
- 了解 RNN 的前向传播过程

## 核心概念

### RNN 的核心思想：记住过去

**RNN（Recurrent Neural Network）= 有记忆的神经网络。**

想象你在读一本小说：
- 读第一章时，你记住了主角的名字
- 读第二章时，你记得第一章发生了什么
- 读第三章时，你记得前两章的剧情

这就是 RNN 做的事：**处理当前输入的同时，记住之前看过的内容。**

### 隐状态：RNN 的"记性"

RNN 用一个向量来存储"记忆"，叫**隐状态（Hidden State）**，通常用 h 表示。

```
处理第1个输入 → 得到隐状态 h₁
处理第2个输入 → 用 h₁ + 第2个输入 → 得到 h₂
处理第3个输入 → 用 h₂ + 第3个输入 → 得到 h₃
...
```

**隐状态会一直传递下去，像接力赛一样。**

### RNN 和普通网络的区别

| 普通网络 | RNN |
|----------|-----|
| 每个输入独立处理 | 输入按顺序处理 |
| 没有记忆 | 有隐状态存储记忆 |
| 输出只依赖当前输入 | 输出依赖当前输入和历史 |

## 原理解析

### RNN 的结构

RNN 的核心公式：

```
hₜ = tanh(Wₓₕ · xₜ + Wₕₕ · hₜ₋₁ + bₕ)
yₜ = Wₕᵧ · hₜ + bᵧ
```

其中：
- xₜ：第 t 个时间步的输入
- hₜ₋₁：上一个时间步的隐状态（记忆）
- hₜ：当前时间步的隐状态
- Wₓₕ：输入到隐状态的权重
- Wₕₕ：隐状态到隐状态的权重（记忆传递）
- yₜ：第 t 个时间步的输出

### 时间展开图解

把 RNN 在时间维度展开：

```
    h₀      h₁      h₂      h₃
    ↓       ↓       ↓       ↓
    ┌───┐   ┌───┐   ┌───┐   ┌───┐
x₁→│RNN│→x₂→│RNN│→x₃→│RNN│→x₄→│RNN│
    └───┘   └───┘   └───┘   └───┘
     ↓       ↓       ↓       ↓
    y₁      y₂      y₃      y₄
```

每个时间步：
1. 接收当前输入 xₜ 和上一个隐状态 hₜ₋₁
2. 计算新的隐状态 hₜ
3. 产生输出 yₜ

### tanh 激活函数的作用

为什么用 tanh 而不是 ReLU？

- tanh 的输出在 [-1, 1] 之间
- 这能让隐状态保持"稳定"，不会无限增长
- 如果用 ReLU，隐状态可能越来越大，导致数值不稳定

### RNN 的三种应用模式

1. **一对多**：一张图片生成描述文字
2. **多对一**：一个句子判断情感（褒/贬）
3. **多对多**：翻译句子、视频分类每一帧

## 代码实战

### RNN 单元实现

```python
import numpy as np


class RNNCell:
    """单个 RNN 单元"""

    def __init__(self, input_size, hidden_size):
        """初始化 RNN 单元

        Args:
            input_size: 输入向量维度
            hidden_size: 隐状态维度
        """
        self.input_size = input_size
        self.hidden_size = hidden_size

        # 初始化权重（使用小随机值）
        self.W_xh = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hh = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_h = np.zeros(hidden_size)

    def forward(self, x, h_prev):
        """前向传播

        Args:
            x: 当前输入，形状 (input_size,)
            h_prev: 上一个隐状态，形状 (hidden_size,)

        Returns:
            h: 新的隐状态，形状 (hidden_size,)
        """
        # 核心公式: h = tanh(W_xh @ x + W_hh @ h_prev + b)
        h = np.tanh(np.dot(x, self.W_xh) + np.dot(h_prev, self.W_hh) + self.b_h)
        return h

    def __call__(self, x, h_prev):
        return self.forward(x, h_prev)


class SimpleRNN:
    """完整的 RNN 网络"""

    def __init__(self, input_size, hidden_size, output_size):
        """初始化 RNN 网络

        Args:
            input_size: 输入向量维度
            hidden_size: 隐状态维度
            output_size: 输出向量维度
        """
        self.hidden_size = hidden_size
        self.rnn_cell = RNNCell(input_size, hidden_size)

        # 输出层权重
        self.W_hy = np.random.randn(hidden_size, output_size) * 0.01
        self.b_y = np.zeros(output_size)

    def forward(self, x_seq):
        """处理整个序列

        Args:
            x_seq: 输入序列，形状 (seq_len, input_size)

        Returns:
            outputs: 所有时间步的输出，形状 (seq_len, output_size)
            hidden_states: 所有隐状态，形状 (seq_len, hidden_size)
        """
        seq_len = x_seq.shape[0]

        # 初始化隐状态为零向量
        h = np.zeros(self.hidden_size)

        outputs = []
        hidden_states = []

        for t in range(seq_len):
            # 更新隐状态
            h = self.rnn_cell(x_seq[t], h)
            hidden_states.append(h.copy())

            # 计算输出
            y = np.dot(h, self.W_hy) + self.b_y
            outputs.append(y.copy())

        return np.array(outputs), np.array(hidden_states)

    def __call__(self, x_seq):
        return self.forward(x_seq)


def softmax(x):
    """Softmax 函数"""
    e_x = np.exp(x - np.max(x))
    return e_x / e_x.sum()


if __name__ == "__main__":
    print("=== RNN 原理演示 ===\n")

    # 设置随机种子以便复现
    np.random.seed(42)

    # 创建 RNN：输入维度 4，隐状态维度 8，输出维度 3
    rnn = SimpleRNN(input_size=4, hidden_size=8, output_size=3)

    # 模拟输入序列：5 个时间步，每个时间步 4 维向量
    seq_len = 5
    x_seq = np.random.randn(seq_len, 4)

    print(f"输入序列形状: {x_seq.shape}")
    print(f"每个时间步输入: {x_seq.shape[1]} 维向量")
    print(f"序列长度: {seq_len} 个时间步\n")

    # 前向传播
    outputs, hidden_states = rnn(x_seq)

    print(f"隐状态形状: {hidden_states.shape}")
    print(f"输出形状: {outputs.shape}\n")

    print("各时间步的隐状态范数（表示记忆强度）:")
    for t in range(seq_len):
        print(f"  时间步 {t}: {np.linalg.norm(hidden_states[t]):.4f}")

    print("\n各时间步的输出（经过 softmax）:")
    for t in range(seq_len):
        probs = softmax(outputs[t])
        print(f"  时间步 {t}: {probs}")

    print("\n=== 关键观察 ===")
    print("1. 每个时间步都有输出和隐状态")
    print("2. 隐状态会随时间传递，记住历史信息")
    print("3. 后面的输出受到前面所有输入的影响")
