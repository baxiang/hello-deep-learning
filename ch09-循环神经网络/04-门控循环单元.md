# 9.4 门控循环单元

## 学习目标

- 理解 GRU 的设计思想
- 掌握 GRU 的两个门控机制
- 了解 GRU 与 LSTM 的区别

## 核心概念

### GRU：LSTM 的简化版

**GRU（Gated Recurrent Unit）是 LSTM 的"精简版"。**

LSTM 有三个门、两条线（细胞状态 + 隐状态），结构比较复杂。

GRU 大胆简化：
- 只有**两个门**（重置门、更新门）
- 只有**一条线**（隐状态）
- 参数更少，计算更快

**但效果和 LSTM 差不多！**

### GRU 的两个门

| 门 | 作用 | 类比 |
|----|------|------|
| 重置门（Reset Gate） | 决定忽略多少过去信息 | "这个新信息重要吗？需要回顾历史吗？" |
| 更新门（Update Gate） | 决定保留多少旧状态 | "直接用新信息，还是混合新旧？" |

### GRU vs LSTM

| 特性 | LSTM | GRU |
|------|------|-----|
| 门数量 | 3 个 | 2 个 |
| 状态数量 | 2 个（C 和 h） | 1 个（h） |
| 参数量 | 多 | 少（约少 1/3） |
| 计算速度 | 慢 | 快 |
| 表达能力 | 强 | 相当 |
| 适用场景 | 复杂任务 | 数据量小、需要快速训练 |

**经验法则：先用 GRU，不够用再换 LSTM。**

## 原理解析

### GRU 的结构

GRU 只有一个隐状态 h，通过两个门来控制信息流动。

### 计算公式

**1. 重置门**

```
rₜ = sigmoid(Wₓᵣ · xₜ + Wₕᵣ · hₜ₋₁ + bᵣ)
```

决定忽略多少过去信息：
- rₜ 接近 0 → 忽略过去，只看当前
- rₜ 接近 1 → 保留过去信息

**2. 更新门**

```
zₜ = sigmoid(Wₓᵤ · xₜ + Wₕᵤ · hₜ₋₁ + bᵤ)
```

决定新旧信息的混合比例：
- zₜ 接近 0 → 保留旧状态
- zₜ 接近 1 → 使用新候选状态

**3. 候选隐状态**

```
h̃ₜ = tanh(Wₓₕ · xₜ + Wₕₕ · (rₜ * hₜ₋₁) + bₕ)
```

注意重置门 rₜ 的作用：如果 rₜ = 0，候选状态完全忽略历史。

**4. 最终隐状态**

```
hₜ = (1 - zₜ) * hₜ₋₁ + zₜ * h̃ₜ
```

更新门控制新旧混合比例。

### 直观理解

GRU 的更新门 zₜ 同时起到了 LSTM 中遗忘门和输入门的作用：
- 当 zₜ = 1 时，相当于 LSTM 的"全部更新"（输入门开，遗忘门关）
- 当 zₜ = 0 时，相当于 LSTM 的"全部保留"（输入门关，遗忘门开）

这就是 GRU 能用更少的门达到类似效果的原因。

## 代码实战

```python
import numpy as np


class GRUCell:
    """GRU 单元实现"""

    def __init__(self, input_size, hidden_size):
        """初始化 GRU 单元

        Args:
            input_size: 输入向量维度
            hidden_size: 隐状态维度
        """
        self.input_size = input_size
        self.hidden_size = hidden_size

        # 重置门权重
        self.W_xr = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hr = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_r = np.zeros(hidden_size)

        # 更新门权重
        self.W_xz = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hz = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_z = np.zeros(hidden_size)

        # 候选隐状态权重
        self.W_xh = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hh = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_h = np.zeros(hidden_size)

    def sigmoid(self, x):
        """Sigmoid 函数"""
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

    def forward(self, x, h_prev):
        """GRU 前向传播

        Args:
            x: 当前输入，形状 (input_size,)
            h_prev: 上一个隐状态，形状 (hidden_size,)

        Returns:
            h: 新的隐状态
        """
        # 重置门：决定忽略多少过去信息
        r = self.sigmoid(np.dot(x, self.W_xr) + np.dot(h_prev, self.W_hr) + self.b_r)

        # 更新门：决定新旧信息的混合比例
        z = self.sigmoid(np.dot(x, self.W_xz) + np.dot(h_prev, self.W_hz) + self.b_z)

        # 候选隐状态
        h_tilde = np.tanh(
            np.dot(x, self.W_xh) + np.dot(r * h_prev, self.W_hh) + self.b_h
        )

        # 最终隐状态：混合旧状态和新候选
        h = (1 - z) * h_prev + z * h_tilde

        return h

    def __call__(self, x, h_prev):
        return self.forward(x, h_prev)


class GRU:
    """完整的 GRU 网络"""

    def __init__(self, input_size, hidden_size, output_size):
        """初始化 GRU 网络"""
        self.hidden_size = hidden_size
        self.gru_cell = GRUCell(input_size, hidden_size)

        # 输出层
        self.W_hy = np.random.randn(hidden_size, output_size) * 0.01
        self.b_y = np.zeros(output_size)

    def forward(self, x_seq):
        """处理整个序列

        Args:
            x_seq: 输入序列，形状 (seq_len, input_size)

        Returns:
            outputs: 所有时间步的输出
            hidden_states: 所有隐状态
        """
        seq_len = x_seq.shape[0]

        h = np.zeros(self.hidden_size)

        outputs = []
        hidden_states = []

        for t in range(seq_len):
            h = self.gru_cell(x_seq[t], h)
            hidden_states.append(h.copy())

            y = np.dot(h, self.W_hy) + self.b_y
            outputs.append(y.copy())

        return np.array(outputs), np.array(hidden_states)

    def __call__(self, x_seq):
        return self.forward(x_seq)


def count_parameters(model):
    """计算模型参数数量"""
    count = 0
    for attr_name in dir(model):
        attr = getattr(model, attr_name)
        if isinstance(attr, np.ndarray):
            count += attr.size
    return count


if __name__ == "__main__":
    print("=== GRU 结构演示 ===\n")

    np.random.seed(42)

    # 创建 GRU
    gru = GRU(input_size=4, hidden_size=8, output_size=3)

    # 模拟输入序列
    seq_len = 5
    x_seq = np.random.randn(seq_len, 4)

    print(f"输入序列: {seq_len} 个时间步，每个 {x_seq.shape[1]} 维")
    print(f"隐状态维度: 8\n")

    # 前向传播
    outputs, h_states = gru(x_seq)

    print("=== 隐状态变化 ===")
    for t in range(seq_len):
        h_norm = np.linalg.norm(h_states[t])
        print(f"  时间步 {t}: 隐状态范数 = {h_norm:.4f}")

    print("\n=== GRU vs LSTM 参数对比 ===")

    # 创建同样大小的 LSTM 和 GRU
    class LSTMForCompare:
        def __init__(self, input_size, hidden_size):
            # LSTM 有 4 组权重（遗忘门、输入门、候选记忆、输出门）
            self.W_x = np.random.randn(input_size, hidden_size * 4)
            self.W_h = np.random.randn(hidden_size, hidden_size * 4)
            self.b = np.zeros(hidden_size * 4)

    class GRUForCompare:
        def __init__(self, input_size, hidden_size):
            # GRU 只有 3 组权重（重置门、更新门、候选状态）
            self.W_x = np.random.randn(input_size, hidden_size * 3)
            self.W_h = np.random.randn(hidden_size, hidden_size * 3)
            self.b = np.zeros(hidden_size * 3)

    lstm_params = count_parameters(LSTMForCompare(4, 8))
    gru_params = count_parameters(GRUForCompare(4, 8))

    print(f"LSTM 参数量: {lstm_params}")
    print(f"GRU 参数量: {gru_params}")
    print(f"GRU 参数约为 LSTM 的 {gru_params / lstm_params * 100:.1f}%")

    print("\n=== 选择建议 ===")
    print("数据量大、追求性能 → LSTM")
    print("数据量小、追求速度 → GRU")
    print("不确定 → 先试 GRU，不够再换 LSTM")
