# 9.3 长短期记忆网络

## 学习目标

- 理解 LSTM 解决的问题
- 掌握 LSTM 的三个门控机制
- 了解 LSTM 的前向传播过程

## 核心概念

### LSTM 解决什么问题？

**普通 RNN 有个致命缺点：记性不好，容易"忘事"。**

想象你在读一本很长的小说：
- 读到第 100 页时，还记得第 1 页主角叫什么吗？
- 普通神经网络记不住太久之前的事
- 这叫**长距离依赖问题**

LSTM（Long Short-Term Memory）就是专门设计来解决这个问题的。

### LSTM 的核心思想：门控机制

LSTM 引入了"门"的概念，可以控制信息的流动：

| 门 | 作用 | 类比 |
|----|------|------|
| 遗忘门 | 决定丢弃哪些旧信息 | 清理大脑中不重要的记忆 |
| 输入门 | 决定存入哪些新信息 | 把重要信息写进大脑 |
| 输出门 | 决定输出哪些信息 | 从大脑中提取当前需要的 |

**就像你的大脑：选择性地遗忘、记忆、回忆。**

### 细胞状态：LSTM 的"主线记忆"

LSTM 除了隐状态 h，还有一个**细胞状态 C**：

- **C（细胞状态）**：长期记忆，变化缓慢
- **h（隐状态）**：短期记忆，每个时间步都变

这就像一条"信息高速公路"，信息可以在上面长距离传输，不会被干扰。

## 原理解析

### LSTM 的结构

LSTM 有两条线：
- **上方**：细胞状态 C（长期记忆）
- **下方**：隐状态 h（短期记忆 + 输出）

### 三个门的详细计算

**1. 遗忘门（Forget Gate）**

```
fₜ = sigmoid(Wₓf · xₜ + Wₕf · hₜ₋₁ + bᶠ)
```

决定从细胞状态中丢弃什么：
- fₜ 接近 0 → 遗忘
- fₜ 接近 1 → 保留

**2. 输入门（Input Gate）**

```
iₜ = sigmoid(Wₓᵢ · xₜ + Wₕᵢ · hₜ₋₁ + bⁱ)        # 哪些信息要存入
c̃ₜ = tanh(Wₓc · xₜ + Wₕc · hₜ₋₁ + bᶜ)          # 新的候选记忆
```

决定往细胞状态中加入什么新信息。

**3. 更新细胞状态**

```
Cₜ = fₜ * Cₜ₋₁ + iₜ * c̃ₜ
```

两部分：遗忘旧记忆 + 加入新记忆。

**4. 输出门（Output Gate）**

```
oₜ = sigmoid(Wₓₒ · xₜ + Wₕₒ · hₜ₋₁ + bᵒ)
hₜ = oₜ * tanh(Cₜ)
```

决定输出什么，输出的内容来自细胞状态。

### 为什么 LSTM 能记住长期信息？

1. **细胞状态是"直达通道"**：信息可以几乎无损地传递
2. **遗忘门可以设为 1**：完全不遗忘，长期保留
3. **梯度传播更顺畅**：解决 RNN 的梯度消失问题

## 代码实战

```python
import numpy as np


class LSTMCell:
    """LSTM 单元实现"""

    def __init__(self, input_size, hidden_size):
        """初始化 LSTM 单元

        Args:
            input_size: 输入向量维度
            hidden_size: 隐状态维度
        """
        self.input_size = input_size
        self.hidden_size = hidden_size

        # 遗忘门权重
        self.W_xf = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hf = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_f = np.zeros(hidden_size)

        # 输入门权重
        self.W_xi = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hi = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_i = np.zeros(hidden_size)

        # 候选记忆权重
        self.W_xc = np.random.randn(input_size, hidden_size) * 0.01
        self.W_hc = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_c = np.zeros(hidden_size)

        # 输出门权重
        self.W_xo = np.random.randn(input_size, hidden_size) * 0.01
        self.W_ho = np.random.randn(hidden_size, hidden_size) * 0.01
        self.b_o = np.zeros(hidden_size)

    def sigmoid(self, x):
        """Sigmoid 函数"""
        return 1 / (1 + np.exp(-np.clip(x, -500, 500)))

    def forward(self, x, h_prev, c_prev):
        """LSTM 前向传播

        Args:
            x: 当前输入，形状 (input_size,)
            h_prev: 上一个隐状态，形状 (hidden_size,)
            c_prev: 上一个细胞状态，形状 (hidden_size,)

        Returns:
            h: 新的隐状态
            c: 新的细胞状态
        """
        # 遗忘门：决定丢弃哪些信息
        f = self.sigmoid(np.dot(x, self.W_xf) + np.dot(h_prev, self.W_hf) + self.b_f)

        # 输入门：决定存入哪些新信息
        i = self.sigmoid(np.dot(x, self.W_xi) + np.dot(h_prev, self.W_hi) + self.b_i)

        # 候选记忆：新的候选值
        c_tilde = np.tanh(np.dot(x, self.W_xc) + np.dot(h_prev, self.W_hc) + self.b_c)

        # 更新细胞状态
        c = f * c_prev + i * c_tilde

        # 输出门：决定输出哪些信息
        o = self.sigmoid(np.dot(x, self.W_xo) + np.dot(h_prev, self.W_ho) + self.b_o)

        # 计算隐状态
        h = o * np.tanh(c)

        return h, c

    def __call__(self, x, h_prev, c_prev):
        return self.forward(x, h_prev, c_prev)


class LSTM:
    """完整的 LSTM 网络"""

    def __init__(self, input_size, hidden_size, output_size):
        """初始化 LSTM 网络"""
        self.hidden_size = hidden_size
        self.lstm_cell = LSTMCell(input_size, hidden_size)

        # 输出层
        self.W_hy = np.random.randn(hidden_size, output_size) * 0.01
        self.b_y = np.zeros(output_size)

    def forward(self, x_seq):
        """处理整个序列

        Args:
            x_seq: 输入序列，形状 (seq_len, input_size)

        Returns:
            outputs: 所有时间步的输出
            hidden_states: 所有隐状态
            cell_states: 所有细胞状态
        """
        seq_len = x_seq.shape[0]

        # 初始化
        h = np.zeros(self.hidden_size)
        c = np.zeros(self.hidden_size)

        outputs = []
        hidden_states = []
        cell_states = []

        for t in range(seq_len):
            h, c = self.lstm_cell(x_seq[t], h, c)
            hidden_states.append(h.copy())
            cell_states.append(c.copy())

            y = np.dot(h, self.W_hy) + self.b_y
            outputs.append(y.copy())

        return np.array(outputs), np.array(hidden_states), np.array(cell_states)

    def __call__(self, x_seq):
        return self.forward(x_seq)


if __name__ == "__main__":
    print("=== LSTM 结构演示 ===\n")

    np.random.seed(42)

    # 创建 LSTM
    lstm = LSTM(input_size=4, hidden_size=8, output_size=3)

    # 模拟输入序列
    seq_len = 5
    x_seq = np.random.randn(seq_len, 4)

    print(f"输入序列: {seq_len} 个时间步，每个 {x_seq.shape[1]} 维")
    print(f"隐状态维度: 8")
    print(f"细胞状态维度: 8（存储长期记忆）\n")

    # 前向传播
    outputs, h_states, c_states = lstm(x_seq)

    print("=== 细胞状态变化（长期记忆）===")
    for t in range(seq_len):
        c_norm = np.linalg.norm(c_states[t])
        print(f"  时间步 {t}: 细胞状态范数 = {c_norm:.4f}")

    print("\n=== 隐状态变化（短期记忆 + 输出）===")
    for t in range(seq_len):
        h_norm = np.linalg.norm(h_states[t])
        print(f"  时间步 {t}: 隐状态范数 = {h_norm:.4f}")

    print("\n=== LSTM vs 普通 RNN ===")
    print("LSTM 优势:")
    print("1. 细胞状态是"高速公路"，信息可以长距离传输")
    print("2. 三个门控可以精确控制记忆")
    print("3. 解决了普通 RNN 的梯度消失问题")
    print("4. 能记住更长时间之前的信息")
