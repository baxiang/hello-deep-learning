# 10.3 自注意力

## 学习目标

- 理解自注意力的核心思想
- 掌握自注意力的计算过程
- 了解位置编码的作用

## 核心概念

### 自注意力：句子内部的注意力

**普通注意力：** Q 来自外部（比如翻译时，Q 来自目标语言）
**自注意力：** Q、K、V 都来自同一个序列（句子自己关注自己）

**自注意力让句子中的每个词都能"看到"其他所有词。**

比如句子"我喜欢深度学习"，处理"学习"这个词时：
- 可以直接关注"我"（主语）
- 可以直接关注"喜欢"（动词）
- 可以直接关注"深度"（修饰词）

每个词都能建立与句子中其他词的直接联系。

### 自注意力的作用

| 作用 | 例子 |
|------|------|
| 消除歧义 | "bank" 是银行还是河岸？看上下文其他词 |
| 建立引用关系 | "他" 指的是谁？找到句子中的人名 |
| 捕获语法结构 | 主谓宾关系、修饰关系 |
| 理解长距离依赖 | 句子开头的主语，句子结尾还能关联到 |

### 位置编码：弥补顺序信息丢失

自注意力有个问题：**它不知道词的顺序。**

计算注意力时，所有词都是平等的：
```
["我", "喜欢", "深度", "学习"]
["学习", "深度", "喜欢", "我"]  # 打乱顺序，注意力结果一样！
```

这显然不对。"我喜欢"和"喜欢我"意思完全不同。

**解决方案：位置编码（Positional Encoding）**

给每个词加上位置信息：
```
词向量 = 原向量 + 位置向量
```

位置向量告诉模型："我是第 1 个词"、"我是第 2 个词"...

## 原理解析

### 自注意力的计算

输入序列 X，生成 Q、K、V：

```
Q = X · W_q
K = X · W_k
V = X · W_v
```

然后计算注意力：

```
Attention(Q, K, V) = softmax(QK^T / √d) · V
```

### 自注意力计算：完整数值演示

**例子：句子 "the cat sat"，计算"sat"的自注意力输出**

```
输入词向量（假设2维，简化）：
  'the' = [1.0, 0.5]
  'cat' = [0.8, 1.2]
  'sat' = [0.3, 0.9]
```

**步骤1：生成 Q、K、V 三组向量**

```
每个词通过权重矩阵生成 Q、K、V：

假设权重矩阵（简化）：
  W_q = [[0.5, 0.3], [0.2, 0.4]]
  W_k = [[0.4, 0.2], [0.3, 0.5]]
  W_v = [[0.6, 0.1], [0.2, 0.7]]

计算 'the' 的 Q、K、V：
  Q_the = [1.0, 0.5] × W_q = [0.5×1.0+0.3×0.5, 0.2×1.0+0.4×0.5]
         = [0.65, 0.40]
  
  K_the = [1.0, 0.5] × W_k = [0.55, 0.40]
  
  V_the = [1.0, 0.5] × W_v = [0.70, 0.45]

计算 'cat' 的 Q、K、V：
  Q_cat = [0.8, 1.2] × W_q = [0.64, 0.56]
  K_cat = [0.8, 1.2] × W_k = [0.56, 0.84]
  V_cat = [0.8, 1.2] × W_v = [0.72, 0.92]

计算 'sat' 的 Q、K、V：
  Q_sat = [0.3, 0.9] × W_q = [0.42, 0.42]
  K_sat = [0.3, 0.9] × W_k = [0.30, 0.51]
  V_sat = [0.3, 0.9] × W_v = [0.27, 0.69]

结果：每个词有3个向量（Q、K、V）
```

**步骤2：计算注意力得分（Score）**

```
计算 'sat' 对所有词的注意力得分：

'sat'的 Q（Query）与每个词的 K（Key）做点积：

得分_the = Q_sat · K_the
          = [0.42, 0.42] · [0.55, 0.40]
          = 0.42×0.55 + 0.42×0.40
          = 0.231 + 0.168
          = 0.399

得分_cat = Q_sat · K_cat
          = [0.42, 0.42] · [0.56, 0.84]
          = 0.235 + 0.353
          = 0.588

得分_sat = Q_sat · K_sat
          = [0.42, 0.42] · [0.30, 0.51]
          = 0.126 + 0.214
          = 0.340

得分结果：
  'sat' 对 'the'：0.399
  'sat' 对 'cat'：0.588  ← 最高，最关注'cat'
  'sat' 对 'sat'：0.340

解释：'sat'最关注'cat'，因为"cat sat"是主谓关系
```

**步骤3：缩放和 Softmax**

```
缩放（除以√d，防止数值过大）：
  d = 2（向量维度）
  √d ≈ 1.414

缩放得分：
  得分_the / √2 = 0.399 / 1.414 ≈ 0.283
  得分_cat / √2 = 0.588 / 1.414 ≈ 0.416
  得分_sat / √2 = 0.340 / 1.414 ≈ 0.240

Softmax（转化成概率）：
  e^(0.283) ≈ 1.327
  e^(0.416) ≈ 1.516
  e^(0.240) ≈ 1.271
  
  总和 = 1.327 + 1.516 + 1.271 ≈ 4.114
  
  权重_the = 1.327 / 4.114 ≈ 0.323
  权重_cat = 1.516 / 4.114 ≈ 0.369  ← 最高权重
  权重_sat = 1.271 / 4.114 ≈ 0.308

注意力权重结果：
  'sat' 对 'the'：32.3%
  'sat' 对 'cat'：36.9%  ← 最关注'cat'
  'sat' 对 'sat'：30.8%
```

**步骤4：加权合并值向量（V）**

```
用注意力权重对 V 向量加权求和：

sat的新向量 = 权重_the × V_the + 权重_cat × V_cat + 权重_sat × V_sat

计算：
  = 0.323 × [0.70, 0.45] + 0.369 × [0.72, 0.92] + 0.308 × [0.27, 0.69]
  
  = [0.323×0.70, 0.323×0.45] + [0.369×0.72, 0.369×0.92] + [0.308×0.27, 0.308×0.69]
  
  = [0.226, 0.145] + [0.266, 0.340] + [0.083, 0.213]
  
  = [0.226+0.266+0.083, 0.145+0.340+0.213]
  
  = [0.575, 0.698]

结果：
  'sat' 的新向量 = [0.575, 0.698]
  
特点：
  融合了 'the'、'cat'、'sat' 的信息
  权重最高是 'cat' → 'sat' 新向量更像 'cat'
  捕捉了"cat sat"的主谓关系
```

**完整流程总结：**

```
输入：'the'、'cat'、'sat' 的词向量

步骤1：生成 Q、K、V
  每个词 → Q向量、K向量、V向量

步骤2：计算注意力得分
  Q_sat · K_the、K_cat、K_sat → 得分

步骤3：Softmax 转权重
  得分 → Softmax → 权重（概率）

步骤4：加权求和
  权重 × V向量 → 新向量

结果：
  'sat' 新向量融合了上下文信息
  重点关注 'cat'（权重最高）
  
意义：
  每个词都这样处理
  最终所有词都融合了上下文
  模型理解句子结构和关系
```

### 自注意力矩阵解读

假设句子有 4 个词，注意力矩阵是 4×4：

```
        词1   词2   词3   词4
词1    0.6   0.2   0.1   0.1    ← 词1 最关注自己
词2    0.1   0.5   0.3   0.1    ← 词2 关注词3（可能是修饰关系）
词3    0.1   0.2   0.6   0.1    ← 词3 最关注自己
词4    0.2   0.1   0.3   0.4    ← 词4 关注词1（可能是引用关系）
```

每一行表示一个词对所有词的关注程度。

### 位置编码的设计

最常用的位置编码（Transformer 原版）：

```
PE(pos, 2i)   = sin(pos / 10000^(2i/d))
PE(pos, 2i+1) = cos(pos / 10000^(2i/d))
```

- pos：位置（第几个词）
- i：维度索引
- d：向量维度

这种编码的特点：
- 不同位置有不同的编码
- 相邻位置的编码相似
- 可以泛化到更长的序列

## 代码实战

```python
import numpy as np


def softmax(x, axis=-1):
    """Softmax 函数"""
    e_x = np.exp(x - np.max(x, axis=axis, keepdims=True))
    return e_x / np.sum(e_x, axis=axis, keepdims=True)


def positional_encoding(seq_len, d_model):
    """位置编码

    Args:
        seq_len: 序列长度
        d_model: 向量维度

    Returns:
        PE: 位置编码矩阵，形状 (seq_len, d_model)
    """
    PE = np.zeros((seq_len, d_model))

    for pos in range(seq_len):
        for i in range(d_model):
            if i % 2 == 0:
                PE[pos, i] = np.sin(pos / (10000 ** (i / d_model)))
            else:
                PE[pos, i] = np.cos(pos / (10000 ** ((i - 1) / d_model)))

    return PE


class SelfAttention:
    """自注意力模块"""

    def __init__(self, d_model):
        """初始化

        Args:
            d_model: 模型维度
        """
        self.d_model = d_model

        # 初始化 QKV 权重
        np.random.seed(42)
        self.W_q = np.random.randn(d_model, d_model) * 0.1
        self.W_k = np.random.randn(d_model, d_model) * 0.1
        self.W_v = np.random.randn(d_model, d_model) * 0.1

    def forward(self, X, mask=None):
        """前向传播

        Args:
            X: 输入序列，形状 (seq_len, d_model)
            mask: 可选的掩码矩阵

        Returns:
            output: 自注意力输出
            weights: 注意力权重矩阵
        """
        seq_len = X.shape[0]

        # 计算 QKV
        Q = np.dot(X, self.W_q)  # (seq_len, d_model)
        K = np.dot(X, self.W_k)
        V = np.dot(X, self.W_v)

        # 计算注意力分数
        scores = np.dot(Q, K.T) / np.sqrt(self.d_model)

        # 应用掩码（可选）
        if mask is not None:
            scores = scores + mask  # mask 中 -inf 的位置会被 softmax 变成 0

        # softmax
        weights = softmax(scores)

        # 加权求和
        output = np.dot(weights, V)

        return output, weights


if __name__ == "__main__":
    print("=" * 50)

    print("=" * 50)

    np.random.seed(42)

    # === 1. 自注意力基本示例 ===
    print("\n=== 1. 自注意力计算 ===")

    # 模拟句子："我 喜欢 深度 学习"
    # 每个词用 8 维向量表示
    seq_len = 4
    d_model = 8

    # 词向量（随机模拟）
    word_vectors = np.random.randn(seq_len, d_model)

    output, weights = self_attn.forward(word_vectors)

    print(f"输入形状: {word_vectors.shape}")
    print(f"输出形状: {output.shape}")
    print(f"权重矩阵形状: {weights.shape}")

    print("\n注意力权重矩阵（每行是一个词对所有词的关注）:")
