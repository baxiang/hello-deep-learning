# 10.2 注意力计算方式

## 学习目标

- 掌握 QKV 框架
- 理解点积注意力和加性注意力
- 了解注意力计算的完整流程

## 核心概念

### QKV 框架：注意力的标准范式

注意力计算用三个向量：

| 向量 | 名称 | 作用 | 类比 |
|------|------|------|------|
| Q | Query（查询） | 当前需要什么信息 | "我想找什么？" |
| K | Key（键） | 每个输入的"标签" | "我是什么类型的信息？" |
| V | Value（值） | 每个输入的实际内容 | "我的具体内容是什么？" |

**类比图书馆检索：**
- Q：你的检索需求（"我要找关于深度学习的书"）
- K：每本书的分类标签（"机器学习"、"历史"、"小说"）
- V：每本书的具体内容

Q 和 K 匹配 → 找到相关书籍 → 取出 V（内容）

### 注意力计算流程

```
1. 计算相关性：Q 和每个 K 的相似度 → 得到 scores
2. 归一化：scores 经过 softmax → 得到 weights
3. 加权求和：weights 和 V 相乘 → 得到 output
```

公式：
```
Attention(Q, K, V) = softmax(Q · K^T) · V
```

### 两种主要计算方式

| 方式 | 公式 | 特点 |
|------|------|------|
| 点积注意力 | softmax(QK^T / √d)V | 计算快，适合大规模 |
| 加性注意力 | softmax(W_qQ + W_kK)V | 表达能力强，适合复杂匹配 |

Transformer 用的是**点积注意力**，因为它计算效率高。

## 原理解析

### 点积注意力详解

```
scores = Q · K^T         # 查询和键的点积，表示相关性
weights = softmax(scores)  # 归一化成概率分布
output = weights · V      # 加权求和
```

**为什么用点积表示相关性？**

点积能衡量两个向量的相似程度：
- 方向相同 → 点积大 → 相关性强
- 方向相反 → 点积小 → 相关性弱
- 垂直 → 点积为 0 → 无关

**为什么要除以 √d？**

当向量维度 d 很大时，点积结果也会变大，softmax 的梯度会变得很小（饱和）。

除以 √d 让点积保持在合理范围，梯度更稳定。

### 加性注意力详解

```
scores = tanh(W_q · Q + W_k · K)  # 通过神经网络学习匹配方式
weights = softmax(W_s · scores)   # 再用一层网络
output = weights · V
```

加性注意力用神经网络学习 Q 和 K 的匹配方式，表达能力更强。

但计算比点积慢（需要更多矩阵运算）。

### 多头注意力的扩展

把注意力分成多个"头"，每个头关注不同的信息：

```
head_1 = Attention(Q_1, K_1, V_1)  # 关注语法关系
head_2 = Attention(Q_2, K_2, V_2)  # 关注语义关系
head_3 = Attention(Q_3, K_3, V_3)  # 关注位置关系
...
output = Concat(head_1, head_2, ...) · W_o
```

就像多个专家分工：有的擅长语法，有的擅长语义，有的擅长识别引用。

## 代码实战

```python
import numpy as np


def softmax(x, axis=-1):
    """Softmax 函数"""
    e_x = np.exp(x - np.max(x, axis=axis, keepdims=True))
    return e_x / np.sum(e_x, axis=axis, keepdims=True)


def scaled_dot_product_attention(Q, K, V):
    """缩放点积注意力

    Args:
        Q: 查询矩阵，形状 (seq_len_q, d_k)
        K: 键矩阵，形状 (seq_len_k, d_k)
        V: 值矩阵，形状 (seq_len_k, d_v)

    Returns:
        output: 注意力输出，形状 (seq_len_q, d_v)
        weights: 注意力权重，形状 (seq_len_q, seq_len_k)
    """
    d_k = Q.shape[-1]

    # 1. 计算相关性分数
    scores = np.dot(Q, K.T) / np.sqrt(d_k)

    # 2. softmax 归一化
    weights = softmax(scores)

    # 3. 加权求和
    output = np.dot(weights, V)

    return output, weights


def additive_attention(Q, K, V, d_hidden=8):
    """加性注意力

    Args:
        Q: 查询矩阵
        K: 键矩阵
        V: 值矩阵
        d_hidden: 隐藏层维度

    Returns:
        output: 注意力输出
        weights: 注意力权重
    """
    seq_len_q = Q.shape[0]
    seq_len_k = K.shape[0]
    d_k = Q.shape[-1]

    # 初始化权重（模拟）
    np.random.seed(42)
    W_q = np.random.randn(d_k, d_hidden) * 0.1
    W_k = np.random.randn(d_k, d_hidden) * 0.1
    W_s = np.random.randn(d_hidden, 1) * 0.1

    # 计算加性分数
    # Q 扩展到 (seq_len_q, seq_len_k, d_k)
    Q_expanded = np.tile(Q[:, np.newaxis, :], (1, seq_len_k, 1))
    K_expanded = np.tile(K[np.newaxis, :, :], (seq_len_q, 1, 1))

    # tanh(W_q·Q + W_k·K)
    hidden = np.tanh(
        np.dot(Q_expanded, W_q) + np.dot(K_expanded, W_k)
    )  # (seq_len_q, seq_len_k, d_hidden)

    # W_s·hidden
    scores = np.dot(hidden, W_s).squeeze(-1)  # (seq_len_q, seq_len_k)

    # softmax
    weights = softmax(scores)

    # 加权求和
    output = np.dot(weights, V)

    return output, weights


if __name__ == "__main__":
    print("=" * 50)

    np.random.seed(42)

    # 模拟数据
    seq_len = 4
    d_k = 8
    d_v = 8

    # Q: 2 个查询（比如要翻译 2 个词）
    Q = np.random.randn(2, d_k)

    # K, V: 4 个输入（原文有 4 个词）
    K = np.random.randn(seq_len, d_k)
    V = np.random.randn(seq_len, d_v)

    print(f"\n输入形状:")
    print(f"  Q: {Q.shape} (2 个查询)")
    print(f"  K: {K.shape} (4 个键)")
    print(f"  V: {V.shape} (4 个值)")

    # === 点积注意力 ===
    print("\n=== 缩放点积注意力 ===")
    output_dot, weights_dot = scaled_dot_product_attention(Q, K, V)

    print(f"输出形状: {output_dot.shape}")
    print(f"权重形状: {weights_dot.shape}")

    print("\n注意力权重矩阵:")
    print("  行=查询，列=输入")

    # === 加性注意力 ===
    print("\n=== 加性注意力 ===")
    output_add, weights_add = additive_attention(Q, K, V)

    print(f"输出形状: {output_add.shape}")
    print("\n注意力权重矩阵:")
    for i, row in enumerate(weights_add):
        print(f"  查询 {i}: {row}")

    # === 对比 ===
    print("\n=== 两种方式对比 ===")
    print("点积注意力:")
    print("  - 计算简单（只有矩阵乘法）")
    print("  - 效率高，适合大规模")
    print("  - Transformer 使用这种")

    print("  - 通过神经网络学习匹配方式")

    print("  - 计算较慢")
