# 6.2 权重初始值

## 学习目标

- 理解权重初始值为什么重要
- 掌握 Xavier 和 He 两种初始化方法
- 理解为什么不能把权重全设为 0

## 核心概念

### 权重初始值影响什么？

权重初始值决定了训练"从哪里开始"。

**类比：** 你要去山顶。
- 起点在山脚 → 路很长，可能半路放弃。
- 起点在半山腰 → 路短一些，更容易到达。

好的初始值让网络学得快，差的初始值让网络根本学不进去。

### 为什么不能全设为 0？

如果所有权重都是 0，所有神经元的输出都一样。

**结果：** 所有神经元"学"一样的东西，等于只有一个神经元在干活。

**类比：** 一个班 30 个学生，老师讲什么他们都抄一样的答案。那这 30 个人和 1 个人没区别。

## 原理解析

### 问题：信号不能太大也不能太小

神经网络有很多层。每一层都会对信号做变换。

**如果权重太大：**
- 第 1 层：信号 × 大数 = 更大的信号
- 第 2 层：更大信号 × 大数 = 巨大信号
- 第 10 层：信号爆炸 → 计算机存不下

**如果权重太小：**
- 第 1 层：信号 × 小数 = 更小的信号
- 第 2 层：更小信号 × 小数 = 微小信号
- 第 10 层：信号变成 0 → 什么都学不到

**这到底意味着什么？**

权重的"大小"（标准差）需要刚刚好，让每一层的信号幅度差不多。

### Xavier 初始值

适用于 Sigmoid 和 Tanh 激活函数。

```
权重 = 随机数 × √(1 / 输入节点数)
```

**直觉：** 输入节点越多，每个节点的权重应该越小。这样加权和不会爆炸。

### He 初始值

适用于 ReLU 激活函数。

```
权重 = 随机数 × √(2 / 输入节点数)
```

**为什么是 2？** 因为 ReLU 会把负数变成 0，相当于"砍掉"了一半的信号。所以要乘以 √2 来补偿。

## 代码实战

### 完整可运行代码

```python
import numpy as np

def relu(x):
    return np.maximum(0, x)

def sigmoid(x):
    return np.where(x >= 0, 1/(1+np.exp(-x)), np.exp(x)/(1+np.exp(x)))

def simulate_signal(input_size, layers, weight_scale, activation):
    """模拟信号通过多层网络"""
    x = np.random.randn(1000, input_size)  # 1000 个样本

    for i, node_count in enumerate(layers):
        W = np.random.randn(x.shape[1], node_count) * weight_scale
        a = np.dot(x, W)
        x = activation(a)

        print(f"  第 {i+1} 层: "
              f"均值 {x.mean():.4f}, "
              f"标准差 {x.std():.4f}, "
              f"0 的比例 {(x == 0).mean():.1%}")

    return x

print("=== 测试不同权重初始值 ===\n")

# 网络: 100 → 100 → 100 → 100
input_size = 100
layers = [100, 100, 100]

print("情况 1: 标准差 = 1（太大）")
print("激活函数: Sigmoid")
x = simulate_signal(input_size, layers, weight_scale=1.0, activation=sigmoid)

print("\n情况 2: 标准差 = 0.01（太小）")
x = simulate_signal(input_size, layers, weight_scale=0.01, activation=sigmoid)

print("\n情况 3: Xavier 标准差 = √(1/100) = 0.1")
x = simulate_signal(input_size, layers, weight_scale=0.1, activation=sigmoid)

print("\n情况 4: ReLU + He 标准差 = √(2/100) ≈ 0.141")
x = simulate_signal(input_size, layers, weight_scale=np.sqrt(2/100), activation=relu)
```

### 运行结果


**解读：**
1. 标准差=1（太大）：第 1 层后所有值都接近 0.5，标准差几乎为 0。信号"死"了。
2. 标准差=0.01（太小）：标准差逐层缩小。信号"消失"了。
3. Xavier：标准差保持在 0.1 左右。信号"健康"。
4. He + ReLU：约一半是 0（ReLU 的作用），其余值分布合理。信号"健康"。

## 避坑指南

1. **激活函数和初始值要匹配** - ReLU 用 He，Sigmoid 用 Xavier。用错了信号会消失或爆炸。

2. **全 0 初始化绝对不行** - 所有神经元学一样的东西。

3. **标准差太大或太小都不行** - 太大 → 信号爆炸；太小 → 信号消失。
