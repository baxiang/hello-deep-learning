# 6.2 权重初始值

## 学习目标

- 理解权重初始值对训练的影响
- 掌握 Xavier 初始值和 He 初始值
- 理解为什么不能把权重全设为 0

## 核心概念

### 为什么权重初始值很重要？

权重初始值决定了训练从哪里开始。好的初始值让网络学得快，差的初始值让网络根本学不进去。

**类比：** 你要去山顶。如果你一开始就在半山腰，你很快能到。如果你一开始在山脚下的坑里，你可能根本爬不出来。

## 原理解析

### 不能用 0 初始化

如果把所有权重视为 0，那么所有神经元的输出都一样，梯度也一样。结果就是 **所有神经元学一样的东西**，等于只有一个神经元在干活。

### Xavier 初始值

适用于激活函数是 Sigmoid 或 Tanh 的网络。

公式：`权重 = 随机数 × sqrt(1 / 输入节点数)`

```python
W = np.random.randn(input_size, output_size) * np.sqrt(1.0 / input_size)
```

### He 初始值

适用于激活函数是 ReLU 的网络。

公式：`权重 = 随机数 × sqrt(2 / 输入节点数)`

```python
W = np.random.randn(input_size, output_size) * np.sqrt(2.0 / input_size)
```

**为什么 He 初始值是 2？** 因为 ReLU 会把负数变成 0，相当于"砍掉"了一半的信号。所以要乘以 sqrt(2) 来补偿。

## 避坑指南

1. **激活函数和初始值要匹配** - ReLU 用 He，Sigmoid 用 Xavier。用错了会导致梯度消失或爆炸。
2. **标准差太大** - 权重初始化得太大，深层网络的输出会越来越大，最后溢出。
3. **标准差太小** - 权重初始化得太小，深层网络的输出会越来越小，最后变成 0。
