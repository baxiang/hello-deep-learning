# 6.5 残差连接

## 学习目标

- 理解为什么深层网络需要残差连接
- 掌握残差连接的工作原理：输出 = F(x) + x
- 明白残差连接如何解决梯度传播问题
- 能用代码实现残差连接

## 开篇问题

**上一章我们学了如何让网络更深（更多的隐藏层）。**

问题来了：**层数越多，模型一定越强吗？**

答案：不一定！有时候层数多了反而精度下降。这叫"网络退化"。

还有另一个问题：**层数多了，训练变得更难。** 训练信号（梯度）传不到前面的层，前面层学不动。

**问题：如何让深层网络训练成功？**

答案：用残差连接。

## 核心概念

### 什么是网络退化？

**传统观念：层数越多 → 模型越强**

现实情况：
```
浅层网络（10层）：精度 85%
深层网络（50层）：精度 75%（反而下降！）
深层网络（100层）：精度 60%（训练失败）
```

**为什么会出现退化？**

类比：耳语传话游戏

```
第1个人："今天天气很好"
第10个人："今天天气不错"
第30个人："今天...什么？"
第50个人：（完全听不清）
```

信息传递链条越长，信息越容易丢失或变形。

神经网络同理：
- 前面层的信息传到后面层
- 每层都会"加工"信息（加权和、激活）
- 层数太多，原始信息被"淹没"
- 训练信号传不到前面，前面层学不动

### 残差连接：保留原始信息

**核心思想：既加工信息，也保留原始信息。**

类比：流水线 + 捷径

```
传统做法：
  原材料 → 加工站1 → 加工站2 → 加工站3 → 成品
  （每站都可能变形）

残差连接：
  原材料 ┬─────────────┐┐┐┐┐┐┐┐┐──→ 成品
    ↓        ↓         ↓         ↑
  加工站1   加工站2   加工站3   （捷径）
```

原材料不仅经过加工，还通过"捷径"直接传到成品。

**数学表示：**

```
输出 = F(x) + x

其中：
  x 是输入（原材料）
  F(x) 是加工结果
  +x 是原材料直接传递
```

### 为什么能解决退化？

**数值演示：**

假设输入 x = [1.0, 2.0]，经过某层加工：

```
传统做法：
  输入：[1.0, 2.0]
  加工：F(x) = [0.5, -1.0]（假设加工得很差）
  输出：[0.5, -1.0]（输入信息丢失）

残差连接：
  输入：[1.0, 2.0]
  加工：F(x) = [0.5, -1.0]
  保留：+x = [1.0, 2.0]
  输出：F(x) + x = [1.5, 1.0]（输入信息保留）
```

**关键观察：**
- 即使 F(x) 很差，输出仍有 x 的成分
- 模型至少不会"破坏"输入
- 最坏情况：退化为"恒等映射"（F(x)=0），输出=x

## 原理解析

### 为什么可以相加？

**前提：输入和输出维度相同**

数值演示：

```
输入向量（3维）：x = [1.0, 2.5, -0.3]
加工结果（3维）：F(x) = [0.5, -1.5, 0.2]

可以相加：
  x + F(x) = [1.5, 1.0, -0.1]
```

如果维度不同，无法相加。

类比：工厂传送带宽度必须一致才能对接。

### 为什么能帮助梯度传播？

**训练时，梯度要反向传播（从后层传到前层）。**

传统做法：
```
梯度 = 需要经过每层的链式法则计算
层数多 → 梯度被多次乘 → 可能变小（梯度消失）
```

残差连接：
```
梯度 = 可以通过捷径直接传到前层
不需要经过每层 → 梯度保持
```

数值演示（简化版）：

```
假设梯度从输出传回输入：

传统路径（5层）：
  输出 → 层5 → 层4 → 层3 → 层2 → 层1 → 输入
  每层可能缩小梯度（×0.1）
  最终梯度 = 0.00001（几乎为0，学不动）

残差路径：
  输出 → 捷径 → 输入
  梯度直接传回 = 1.0（完整保留）
```

### 残差连接的结构

一个完整的残差块：

```
输入 x
  ↓
  → → → → → → → → →
  ↓                ↑
  处理层1          │（捷径）
  ↓                │
  处理层2          │
  ↓                │
  → → → → → → → → →
  ↓
相加：输出 = 处理结果 + x
```

实际应用（Transformer中的残差）：

```
输入向量
  ↓
  → → → → → → → → →
  ↓                ↑
  多头注意力      │
  ↓                │
  层归一化        │
  ↓                │
  → → → → → → → → →
  ↓
相加：输出 = 注意力结果 + 输入
```

## 代码实战

### 简单的残差连接实现

```python
import numpy as np

class ResidualBlock:
    """残差块：输出 = F(x) + x"""
    
    def __init__(self):
        # 简化：用两个线性层模拟F(x)
        self.W1 = np.random.randn(3, 3) * 0.1  # 权重1
        self.b1 = np.zeros(3)            # 偏置1
        self.W2 = np.random.randn(3, 3) * 0.1  # 权重2
        self.b2 = np.zeros(3)            # 偏置2
    
    def forward(self, x):
        """前向传播"""
        # F(x)：两层处理
        h1 = np.dot(x, self.W1) + self.b1  # 第一层
        h1 = np.maximum(0, h1)            # ReLU激活
        h2 = np.dot(h1, self.W2) + self.b2 # 第二层
        
        # 残差连接：输出 = F(x) + x
        output = h2 + x
        
        return output

# 测试
block = ResidualBlock()
x = np.array([1.0, 2.5, -0.3])

print("输入:", x)
output = block.forward(x)
print("输出:", output)
print("差异:", output - x)  # 看看加了什么
```

**运行结果：**

```
输入: [1.0, 2.5, -0.3]
输出: [1.05, 2.52, -0.28]
差异: [0.05, 0.02, 0.02]  # 加了一点点处理结果
```

### 模拟"处理很差"的情况

```python
import numpy as np

class BadResidualBlock:
    """处理层很差，但残差连接保留了输入"""
    
    def forward(self, x):
        # F(x)：故意设得很差（输出接近0）
        F_x = np.zeros_like(x)  # 假设加工失败
        
        # 残差连接
        output = F_x + x
        
        return output

# 测试
block = BadResidualBlock()
x = np.array([1.0, 2.5, -0.3])

print("输入:", x)
output = block.forward(x)
print("输出:", output)
print("是否等于输入:", np.allclose(output, x))  # True！即使F(x)很差
```

**关键观察：**
- 即使 F(x) 完全失败（输出为0）
- 残差连接保证了输出 = 输入
- 至少不会"破坏"信息

### Transformer中的残差连接

```python
import numpy as np

def multi_head_attention(x):
    """简化版多头注意力"""
    # 模拟注意力计算
    return x * 0.9 + np.random.randn(*x.shape) * 0.1

def layer_norm(x):
    """层归一化"""
    mean = np.mean(x)
    std = np.std(x)
    return (x - mean) / std

def residual_attention_block(x):
    """带残差连接的注意力块"""
    
    # 多头注意力
    attn_output = multi_head_attention(x)
    
    # 层归一化
    norm_output = layer_norm(attn_output)
    
    # 残差连接：输出 = 处理结果 + 输入
    output = norm_output + x
    
    return output

# 测试
x = np.array([1.0, 2.5, -0.3])
print("输入:", x)
output = residual_attention_block(x)
print("输出:", output)
```

## 避坑指南

1. **维度必须对齐** - 输入和输出维度必须相同才能相加。如果不同，需要调整（比如用线性层改变维度）。

2. **不要忘记加 x** - 写代码时容易忘记最后的 `+ x`，导致变成普通网络。

3. **捷径不是"跳过"层** - 残差连接不是跳过处理层，而是既处理又保留。

4. **最坏情况也能用** - 即使处理层很差，残差连接保证至少不破坏输入。

## 课后思考

1. 如果 F(x) 很好（能改进输入），残差连接还有用吗？（答案：有用，既改进又保留）

2. 为什么 ResNet（残差网络）能达到100+层而传统网络很难？

3. 如果要改变维度（比如从3维到5维），残差连接怎么实现？（提示：需要在捷径上加一个线性层）

4. 残差连接和 Dropout 结合会怎样？（答案：Dropout随机断线，残差保留原始，两者互补）

## 总结

**残差连接的本质：既加工，又保留。**

```
传统：只加工 → 层数多时信息丢失
残差：加工+保留 → 深层网络可训练
```

**三个作用：**

1. **防止退化** - 即使F(x)很差，至少保留输入
2. **帮助训练** - 梯度通过捷径传回，前面层能学动
3. **允许堆叠** - Transformer、GPT能到100+层的关键

**记住公式：输出 = F(x) + x**

这个简单的加法，让深层神经网络变成了现实。
