# 5.2 链式法则

## 学习目标

- 理解链式法则的本质
- 掌握 ReLU 和 Sigmoid 的反向传播
- 能用代码实现这些层的反向传播

## 核心概念

### 什么是链式法则？

链式法则回答一个问题：**如果 A 影响 B，B 影响 C，那么 A 怎么影响 C？**

答案很简单：**把中间每一步的影响乘起来**。

**类比：传话游戏**

A 对 B 说话，声音放大 3 倍。
B 对 C 说话，声音放大 2 倍。

那 A 对 C 的影响 = 3 × 2 = 6 倍。

**这就是链式法则：复合函数的导数 = 各段导数的乘积。**

**公式（不用背）：**

```
如果 y = f(g(x))
那么 dy/dx = dy/du × du/dx   （其中 u = g(x)）
```

**这到底意味着什么？**

想知道 x 对 y 的影响，先算 x 对中间值 u 的影响，再算 u 对 y 的影响，乘起来就行。

## 原理解析

### ReLU 层的反向传播

ReLU 函数：输入 > 0 时输出原值，输入 ≤ 0 时输出 0。

**正向传播：**
- 输入 3 → 输出 3
- 输入 -2 → 输出 0

**反向传播：**
- 如果正向传播时输入 > 0 → 导数原样通过（×1）
- 如果正向传播时输入 ≤ 0 → 导数变成 0（×0）

**类比：** ReLU 像一个"门"。
- 正向时门开了（输入 > 0）→ 反向时信号能通过。
- 正向时门关了（输入 ≤ 0）→ 反向时信号被拦住。

### Sigmoid 层的反向传播

Sigmoid 函数：把任意数字压缩到 0~1 之间。

**正向传播：**
- 输入 0 → 输出 0.5
- 输入 5 → 输出 ≈ 1
- 输入 -5 → 输出 ≈ 0

**反向传播：**
Sigmoid 的导数有个很漂亮的公式：

```
导数 = 输出 × (1 - 输出)
```

只需要知道正向传播的输出，就能算导数。

**这到底意味着什么？**

- 输出 = 0.5 时，导数 = 0.5 × 0.5 = 0.25（最大）
- 输出 = 0.9 时，导数 = 0.9 × 0.1 = 0.09（变小了）
- 输出 = 0.99 时，导数 = 0.99 × 0.01 = 0.009（更小了）

**这意味着：** Sigmoid 的输出越接近 0 或 1，导数越小，网络学得越慢。这就是"梯度消失"问题。

**类比：** 一个老师，学生考 50 分（输出 0.5）时他最有热情教（导数最大）；学生考 99 分或 1 分时他反而不知道怎么教了（导数很小）。

## 代码实战

### 完整可运行代码

```python
import numpy as np

# === 1. ReLU 层 ===
class ReluLayer:
    def __init__(self):
        # 记录哪些位置输入 ≤ 0（门是关着的）
        self.mask = None

    def forward(self, x):
        """正向传播：正数通过，负数变 0"""
        self.mask = (x <= 0)   # 记录负数位置
        out = x.copy()
        out[self.mask] = 0     # 负数变 0
        return out

    def backward(self, dout):
        """反向传播：负数位置导数变 0"""
        dout[self.mask] = 0    # 关着门的地方导数传不过去
        return dout

# === 2. Sigmoid 层 ===
class SigmoidLayer:
    def __init__(self):
        self.out = None

    def forward(self, x):
        """正向传播：压缩到 0~1"""
        out = 1 / (1 + np.exp(-x))
        self.out = out
        return out

    def backward(self, dout):
        """反向传播：导数 = 输出 × (1 - 输出)"""
        return dout * self.out * (1 - self.out)

# === 3. 测试 ===
print("=== ReLU 层测试 ===")
relu = ReluLayer()
x = np.array([3.0, -2.0, 0.0, 5.0, -1.0])
print(f"输入: {x}")

y = relu.forward(x)
print(f"正向输出: {y}")
print("  → 正数不变，负数变 0")

dy = relu.backward(np.ones_like(y))
print(f"反向输出: {dy}")
print("  → 原来正数的位置导数=1，负数的位置导数=0")

print("\n=== Sigmoid 层测试 ===")
sigmoid = SigmoidLayer()
x = np.array([-2.0, 0.0, 2.0])
print(f"输入: {x}")

y = sigmoid.forward(x)
print(f"正向输出: {y}")
print("  → 都在 0~1 之间")

dy = sigmoid.backward(np.ones_like(y))
print(f"反向输出: {dy}")
print("  → 验证: 输出 × (1-输出) = 导数")
for i in range(3):
    expected = y[i] * (1 - y[i])
    print(f"    y[{i}]={y[i]:.4f}, y×(1-y)={expected:.4f}, 导数={dy[i]:.4f}")
```

### 运行结果

```
=== ReLU 层测试 ===
输入: [ 3. -2.  0.  5. -1.]
正向输出: [3. 0. 0. 5. 0.]
  → 正数不变，负数变 0
反向输出: [1. 0. 0. 1. 0.]
  → 原来正数的位置导数=1，负数的位置导数=0

=== Sigmoid 层测试 ===
输入: [-2.  0.  2.]
正向输出: [0.11920292 0.5        0.88079708]
  → 都在 0~1 之间
反向输出: [0.10499359 0.25       0.10499359]
  → 验证: 输出 × (1-输出) = 导数
    y[0]=0.1192, y×(1-y)=0.1050, 导数=0.1050
    y[1]=0.5000, y×(1-y)=0.2500, 导数=0.2500
    y[2]=0.8808, y×(1-y)=0.1050, 导数=0.1050
```

**解读：**
1. ReLU：正向时"开门"（正数）的位置，反向时导数能通过；"关门"（负数）的位置，导数被拦住。
2. Sigmoid：输出越接近 0.5，导数越大；越接近 0 或 1，导数越小。

### 逐行解读

**ReLU 正向：**
```python
self.mask = (x <= 0)   # 记录哪些位置是负数
out = x.copy()
out[self.mask] = 0     # 负数变 0
```
1. `self.mask` 是一个布尔数组，记录哪些位置 `x ≤ 0`。
2. `x.copy()` 复制一份，不修改原始数据。
3. 把负数位置设为 0。

**ReLU 反向：**
```python
dout[self.mask] = 0
return dout
```
1. 把负数位置的导数设为 0（门关了，信号传不过去）。
2. 正数位置的导数原样返回。

**Sigmoid 反向：**
```python
return dout * self.out * (1 - self.out)
```
1. `self.out` 是正向传播的输出。
2. 导数 = 上游导数 × 输出 × (1 - 输出)。

## 避坑指南

1. **ReLU 必须保存 mask** - 反向传播需要知道哪些位置是负数。如果没保存，就不知道该把哪些导数设为 0。

2. **Sigmoid 必须保存输出** - 反向传播需要用到正向的输出值。

3. **`x.copy()` 不能省** - 如果直接修改 x，会影响原始数据。
