# 5.2 链式法则

## 学习目标

- 理解链式法则的本质
- 掌握激活函数（ReLU, Sigmoid）的反向传播
- 理解为什么链式法则是反向传播的核心

## 核心概念

### 什么是链式法则？

链式法则是微积分中的一个规则，告诉我们 **复合函数的导数怎么算**。

**大白话版本：**

假设你有两条路：A → B → C。
- A 到 B 的变化率是 3（A 变 1，B 变 3）。
- B 到 C 的变化率是 2（B 变 1，C 变 2）。

那么 A 到 C 的变化率是 3 × 2 = 6。

**这就是链式法则：复合函数的导数 = 各段导数的乘积。**

### 为什么需要链式法则？

神经网络是由很多层函数 **嵌套** 在一起的：

```
输出 = f(g(h(x)))
```

要对 x 求导，如果直接展开再求导，式子会变得极其复杂。链式法则告诉我们：**只需要逐段求导，然后乘起来就行**。

## 原理解析

### ReLU 层的反向传播

ReLU 函数：`y = x (x > 0), y = 0 (x ≤ 0)`

导数：
- `x > 0` 时，导数 = 1。
- `x ≤ 0` 时，导数 = 0。

反向传播：
- 如果正向传播时输入 `x > 0`，上游导数 **原样通过**。
- 如果 `x ≤ 0`，导数被"拦住"，变成 0。

**形象理解：** ReLU 就像一个"单向门"，正向时只让正数通过，反向时也只让正数位置的导数通过。

### Sigmoid 层的反向传播

Sigmoid 函数：`y = 1 / (1 + e^(-x))`

导数公式：`dy/dx = y × (1 - y)`

反向传播：只需要知道正向传播的输出 `y`，就能算出导数。

**为什么这很方便？** 因为在神经网络的正向传播中，我们已经计算并保存了 `y`，反向传播时直接用就行。

**形象理解：** Sigmoid 就像一个"过滤器"，不管上游传来多大的导数，经过 Sigmoid 后都会被压缩。当 y = 0.5 时导数最大（0.25），y 接近 0 或 1 时导数接近 0。

## 代码实战

```python
import numpy as np

class ReluLayer:
    """ReLU 层"""

    def __init__(self):
        self.mask = None

    def forward(self, x):
        self.mask = (x <= 0)   # 记录哪些位置 x ≤ 0
        out = x.copy()
        out[self.mask] = 0     # x ≤ 0 的地方输出 0
        return out

    def backward(self, dout):
        dout[self.mask] = 0    # x ≤ 0 的地方导数为 0
        dx = dout
        return dx

class SigmoidLayer:
    """Sigmoid 层"""

    def __init__(self):
        self.out = None

    def forward(self, x):
        out = 1 / (1 + np.exp(-x))
        self.out = out
        return out

    def backward(self, dout):
        dx = dout * self.out * (1 - self.out)
        return dx

# === 测试 ReLU 层 ===
print("=== ReLU 层 ===")
relu = ReluLayer()
x = np.array([[1, -2], [-3, 4]])
y = relu.forward(x)
print("ReLU 正向输入:", x)
print("ReLU 正向输出:", y)         # [[1, 0], [0, 4]]
dy = relu.backward(np.ones_like(y))
print("ReLU 反向输出:", dy)        # [[1, 0], [0, 1]]

# === 测试 Sigmoid 层 ===
print("\n=== Sigmoid 层 ===")
sigmoid = SigmoidLayer()
y = sigmoid.forward(x)
print("Sigmoid 正向输入:", x)
print("Sigmoid 正向输出:", y)
dy = sigmoid.backward(np.ones_like(y))
print("Sigmoid 反向输出:", dy)

# === 验证链式法则 ===
print("\n=== 链式法则验证 ===")
# 假设有一个复合函数: f(x) = sigmoid(relu(x))
# 当 x = [[1, -2], [-3, 4]] 时
# 先过 ReLU: [[1, 0], [0, 4]]
# 再过 Sigmoid: [[0.731, 0.5], [0.5, 0.982]]

# 反向传播：从后往前
dout = np.ones_like(y)  # 假设最终导数为 1
dx_sigmoid = sigmoid.backward(dout)
print("经过 Sigmoid 反向后的导数:", dx_sigmoid)
# 再经过 ReLU 反向（需要用到 ReLU 的 mask）
dx_relu = relu.backward(dx_sigmoid)
print("经过 ReLU 反向后的导数（最终）:", dx_relu)
print("原为负数的位置导数为 0，正数位置的导数被保留")
```

**代码解读：**
- `ReluLayer` 用 `mask` 记录哪些位置 `x ≤ 0`，反向传播时把这些位置的导数设为 0。
- `SigmoidLayer` 保存正向传播的输出 `self.out`，反向传播时用公式 `y × (1 - y)` 计算导数。
- 链式法则验证：复合函数 `sigmoid(relu(x))` 的反向传播就是依次经过 Sigmoid 反向和 ReLU 反向，每段导数相乘。
- `x.copy()` 避免修改原始输入数据。

## 避坑指南

1. **copy 的重要性** - `ReluLayer.forward` 中用 `x.copy()` 而不是直接修改 x，否则会影响原始数据。
2. **Sigmoid 的数值稳定** - 当 x 很大时，`np.exp(-x)` 接近 0，没问题。但当 x 是很大的负数时，`np.exp(-x)` 会溢出。生产环境需要注意。
3. **mask 是布尔数组** - `self.mask = (x <= 0)` 返回的是布尔值数组，可以直接用作索引。
