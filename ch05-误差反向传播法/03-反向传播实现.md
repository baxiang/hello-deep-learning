# 5.3 完整的反向传播实现

## 学习目标

- 理解 Affine 层（矩阵乘法层）的反向传播
- 理解 Softmax-with-Loss 层的反向传播
- 把各层组合起来，实现完整的神经网络

## 核心概念

### Affine 层是什么？

Affine 层就是神经网络里的"矩阵乘法 + 偏置"。

```
输出 = 输入 × 权重 + 偏置
```

这个名字听起来复杂，其实就是我们在第 3 章学的矩阵运算。

**类比：** 一个加工厂，原料是输入，加工配方是权重，产出是输出。

### Softmax-with-Loss 层是什么？

这是输出层的"组合拳"：
1. 先用 Softmax 把输出变成概率
2. 再用交叉熵计算损失

把它们合在一层，因为**反向传播的结果特别简洁**：

```
反向传播输出 = (预测概率 - 正确答案) / 批次大小
```

**这到底意味着什么？**

你不需要单独算 Softmax 的导数再算交叉熵的导数。合在一起，导数就是"预测减答案"，简单到不可思议。

**类比：** 你知道考试错了哪些题（正确答案），也知道你答了什么（预测）。"预测 - 答案"就是你需要改进的地方。

## 原理解析

### Affine 层的反向传播

正向：`Y = X · W + B`

反向需要算三个导数：
- **dX**（输入的导数）= `dY · W转置`
- **dW**（权重的导数）= `X转置 · dY`
- **dB**（偏置的导数）= `dY 沿批次求和`

**为什么 dX 要乘 W 的转置？**

因为正向是 `X · W`，反向就要反过来 `· W转置`。就像你从 A 走到 B 用了某条路，从 B 回到 A 要走反方向。

**维度变化：**

```
正向: X(批次, 输入) · W(输入, 输出) → Y(批次, 输出)

反向: dY(批次, 输出) · W转置(输出, 输入) → dX(批次, 输入)
      X转置(输入, 批次) · dY(批次, 输出) → dW(输入, 输出)
```

### Softmax-with-Loss 的反向传播

这是深度学习能高效训练的关键。

```
正向: 输入 a → Softmax → 概率 y → 交叉熵 → 损失 E

反向: 损失 E 的导数(1) → Softmax-with-Loss → (y - t) / batch_size
```

**y - t 的含义：**

- y 是预测概率，t 是正确答案（独热编码）。
- `y - t` 就是"预测错了多少"。
- 预测对的类别，`y - t` 是正数（应该增加权重）。
- 预测错的类别，`y - t` 是负数（应该减少权重）。

## 代码实战

### 完整可运行代码

```python
import numpy as np

# === 1. Affine 层 ===
class AffineLayer:
    """矩阵乘法层"""
    def __init__(self, W, b):
        self.W = W
        self.b = b
        self.x = None
        self.dW = None
        self.db = None

    def forward(self, x):
        """正向传播: Y = X · W + B"""
        self.x = x
        return np.dot(x, self.W) + self.b

    def backward(self, dout):
        """反向传播"""
        dx = np.dot(dout, self.W.T)    # dX = dY · W转置
        self.dW = np.dot(self.x.T, dout)  # dW = X转置 · dY
        self.db = np.sum(dout, axis=0)    # dB = dY 求和
        return dx

# === 2. Softmax-with-Loss 层 ===
class SoftmaxWithLossLayer:
    """Softmax + 交叉熵"""
    def __init__(self):
        self.loss = None
        self.y = None
        self.t = None

    def forward(self, x, t):
        """正向传播"""
        self.t = t
        self.y = softmax(x)
        self.loss = cross_entropy_error(self.y, self.t)
        return self.loss

    def backward(self, dout=1):
        """反向传播: (预测 - 答案) / 批次大小"""
        batch_size = self.t.shape[0]
        dx = (self.y - self.t) / batch_size
        return dx

# === 辅助函数 ===
def softmax(a):
    """Softmax 函数"""
    c = np.max(a)
    exp_a = np.exp(a - c)
    return exp_a / np.sum(exp_a)

def cross_entropy_error(y, t):
    """交叉熵误差"""
    delta = 1e-7
    return -np.sum(t * np.log(y + delta))

# === 3. 测试 Affine 层 ===
print("=== Affine 层测试 ===")

# 创建 Affine 层: 2 输入 → 2 输出
W = np.array([[0.1, 0.2],
              [0.3, 0.4]])
b = np.array([0.1, 0.2])
affine = AffineLayer(W, b)

# 输入: 1 个样本，2 个特征
x = np.array([[1.0, 2.0]])
out = affine.forward(x)
print(f"输入 x: {x}")
print(f"输出: {out}")
print(f"  → 手算: [1×0.1+2×0.3+0.1, 1×0.2+2×0.4+0.2] = [0.8, 1.2]")

# 反向传播
dout = np.array([[1.0, 1.0]])
dx = affine.backward(dout)
print(f"\ndout: {dout}")
print(f"dX: {dx}")
print(f"dW: {affine.dW}")
print(f"dB: {affine.db}")

# === 4. 测试 Softmax-with-Loss ===
print("\n=== Softmax-with-Loss 测试 ===")

# 假设网络输出（3 个类别的概率）
a = np.array([[0.3, 0.3, 0.4]])
# 正确答案: 第 3 个类别（索引 2）
t = np.array([[0.0, 0.0, 1.0]])

layer = SoftmaxWithLossLayer()
loss = layer.forward(a, t)
print(f"网络输出: {a}")
print(f"正确答案: {t}")
print(f"Softmax 概率: {layer.y}")
print(f"损失: {loss:.4f}")

# 反向传播
dx = layer.backward()
print(f"\n反向传播输出 (预测 - 答案): {dx}")
print(f"  → 第 1 类: {dx[0][0]:.4f} (预测偏多，应该减少)")
print(f"  → 第 2 类: {dx[0][1]:.4f} (预测偏多，应该减少)")
print(f"  → 第 3 类: {dx[0][2]:.4f} (预测偏少，应该增加)")
```

### 逐行解读

**Affine 正向：**
```python
def forward(self, x):
    self.x = x
    return np.dot(x, self.W) + self.b
```
1. 保存输入 x（反向时用）。
2. 矩阵乘法 + 偏置。

**Affine 反向：**
```python
dx = np.dot(dout, self.W.T)
self.dW = np.dot(self.x.T, dout)
self.db = np.sum(dout, axis=0)
```
1. `dX = dY · W转置`：把误差"分配"回输入。
2. `dW = X转置 · dY`：计算权重该怎么调。
3. `dB = dY 求和`：偏置没有批次维度，所以要沿批次求和。

**Softmax-with-Loss 反向：**
```python
batch_size = self.t.shape[0]
dx = (self.y - self.t) / batch_size
```
1. `batch_size` 是批次大小（样本数）。
2. `y - t` 就是"预测减答案"。
3. 除以 `batch_size` 是为了让不同批次大小的损失可比。

## 避坑指南

1. **矩阵转置别搞反** - `dW = X转置 · dY`，不是 `dY · X转置`。维度要对上。

2. **偏置求和** - `dB = np.sum(dout, axis=0)`。因为偏置没有批次维度，要把批次方向的误差加起来。

3. **Softmax-with-Loss 的简洁性** - 不要分开算 Softmax 和交叉熵的导数。合在一起就是 `(y - t) / batch_size`。
