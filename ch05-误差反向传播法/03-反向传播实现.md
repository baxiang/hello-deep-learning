# 5.3 完整的反向传播实现

## 学习目标

- 把之前学到的层组合起来，实现完整的神经网络
- 掌握 Affine 层（矩阵乘法层）的反向传播
- 理解 Softmax-with-Loss 层的实现

## 核心概念

### 什么是 Affine 层？

Affine 层就是神经网络中 **矩阵乘法 + 偏置** 的那一步：

```
Y = X · W + B
```

- X 是输入数据
- W 是权重
- B 是偏置

在正向传播中，我们计算这个公式。在反向传播中，我们需要计算 X、W、B 各自的导数。

### Softmax-with-Loss 层

这是输出层的"组合拳"：
1. 先用 Softmax 把输出转换成概率。
2. 再用交叉熵计算损失。

把它们合在一个层里，因为 **Softmax + 交叉熵的反向传播结果非常简单**：

```
反向传播的输出 = (预测概率 - 正确答案) / 批次大小
```

这个结果太简洁了，是深度学习能高效训练的关键。

## 原理解析

### Affine 层的反向传播

正向：`Y = X · W + B`

反向：
- `dX = dY · W^T` （dY 乘以 W 的转置）
- `dW = X^T · dY` （X 的转置乘以 dY）
- `dB = sum(dY, axis=0)` （沿批次方向求和）

**类比：** 正向传播是"信号往前传"，反向传播是"误差往回传"。Affine 层的反向传播就是把误差按权重分配回去。

### Softmax-with-Loss 为什么这么简洁？

单独看：
- Softmax 的反向传播比较复杂。
- 交叉熵的反向传播也比较复杂。

但 **两者组合在一起**，中间的复杂项恰好抵消了，最终只剩下 `(y - t) / batch_size`。这就是为什么深度学习框架中这两个操作通常绑定在一起。

## 代码实战

```python
import numpy as np

class AffineLayer:
    """矩阵乘法层（Affine 层）"""

    def __init__(self, W, b):
        self.W = W
        self.b = b
        self.x = None
        self.dW = None
        self.db = None

    def forward(self, x):
        self.x = x
        out = np.dot(x, self.W) + self.b
        return out

    def backward(self, dout):
        dx = np.dot(dout, self.W.T)
        self.dW = np.dot(self.x.T, dout)
        self.db = np.sum(dout, axis=0)
        return dx

class SoftmaxWithLossLayer:
    """Softmax + 交叉熵损失层"""

    def __init__(self):
        self.loss = None
        self.y = None
        self.t = None

    def forward(self, x, t):
        self.t = t
        self.y = softmax(x)
        self.loss = cross_entropy_error(self.y, self.t)
        return self.loss

    def backward(self, dout=1):
        batch_size = self.t.shape[0]
        dx = (self.y - self.t) / batch_size
        return dx

def softmax(a):
    c = np.max(a)
    exp_a = np.exp(a - c)
    return exp_a / np.sum(exp_a)

def cross_entropy_error(y, t):
    delta = 1e-7
    return -np.sum(t * np.log(y + delta))

# === 测试 Affine 层 ===
print("=== Affine 层测试 ===")
W = np.array([[0.1, 0.2], [0.3, 0.4]])
b = np.array([0.1, 0.2])
affine = AffineLayer(W, b)

x = np.array([[1.0, 2.0]])
out = affine.forward(x)
print("Affine 正向输入:", x)
print("Affine 正向输出:", out)  # [1.0*0.1+2.0*0.3+0.1, 1.0*0.2+2.0*0.4+0.2] = [0.8, 1.2]

dout = np.array([[1.0, 1.0]])
dx = affine.backward(dout)
print("Affine 反向 dx:", dx)
print("Affine 反向 dW:", affine.dW)
print("Affine 反向 db:", affine.db)

# === 测试 SoftmaxWithLoss 层 ===
print("\n=== SoftmaxWithLoss 层测试 ===")
softmax_loss = SoftmaxWithLossLayer()

# 模拟 3 分类问题，批次大小 2
x = np.array([[1.0, 2.0, 3.0],
              [1.0, 2.0, 3.0]])
# one-hot 标签：第一个样本是第 2 类，第二个样本是第 0 类
t = np.array([[0, 1, 0],
              [1, 0, 0]])

loss = softmax_loss.forward(x, t)
print("损失值:", loss)

dx = softmax_loss.backward()
print("反向传播输出:", dx)
print("形状:", dx.shape)  # (2, 3)，和输入 x 一样
print("第一行（预测 - 真实）/ 2:", dx[0])
```

**代码解读：**
- `AffineLayer` 保存了正向传播的输入 `self.x`，反向传播时用来计算 dW 和 dx。
- `self.dW` 和 `self.db` 作为实例变量保存，供优化器后续使用。
- `SoftmaxWithLossLayer` 的反向传播极其简洁：`(预测 - 真实) / 批次大小`。这就是为什么 Softmax 和交叉熵要放在一起。
- `np.dot(dout, self.W.T)` 中 `W.T` 是 W 的转置，矩阵乘法反向传播的关键。
- `softmax` 函数中减去 `np.max(a)` 是为了防止 `np.exp` 溢出。

## 避坑指南

1. **转置别搞反** - `dW = X^T · dY`，不是 `dY · X^T`。维度要匹配。
2. **偏置的梯度** - 偏置 b 没有批次维度，反向传播时需要 `sum(axis=0)` 把批次方向的梯度加起来。
3. **Softmax-with-Loss 的简洁性** - 不要单独算 Softmax 的反向传播再算交叉熵的反向传播，合在一起才是 `(y - t) / batch_size`。
4. **Softmax 的数值稳定** - 正向传播中 `exp(a - max(a))` 防止溢出，这一步不能省。
