# 3.1 神经网络基础

## 学习目标

- 理解神经网络和感知机的区别
- 掌握神经网络的三层结构（输入层、隐藏层、输出层）
- 理解激活函数的作用
- 能用代码跑通一个最简单的神经网络

## 核心概念

### 神经网络 vs 感知机：开关 vs 旋钮

上一章的感知机，输出只有两种可能：0 或 1。

**类比：** 感知机像一个电灯开关，只有"开"和"关"。

神经网络不一样，它的输出可以是 0.3、0.7、0.99…… 任意中间值。

**类比：** 神经网络像一个调光旋钮，可以从暗到亮平滑调节。

**为什么这个区别很重要？**

- 开关只能做"是或否"的判断（比如"这是猫吗？"）。
- 旋钮能做"有多像"的判断（比如"这张图 70% 像猫，30% 像狗"）。

现实世界中，很多事情不是非黑即白的。神经网络能表达"灰度"，所以它能处理更复杂的任务。

### 神经网络的三层结构

神经网络分三层，像一条流水线：

```
输入层 → 隐藏层 → 输出层
  ↓         ↓         ↓
收数据    加工数据   出结果
```

**1. 输入层（收件员）**

负责接收原始数据。比如识别手写数字，输入层接收的就是图片的每个像素值。

**2. 隐藏层（加工厂）**

这是神经网络的核心。它负责把原始数据"加工"成有用的信息。

**类比：** 就像你收到一袋面粉（输入），隐藏层是面包店，把面粉做成面包（有用的信息）。

**3. 输出层（快递员）**

把加工好的结果送出来。比如识别手写数字，输出层会告诉你"这是数字 3 的概率是 85%"。

### 命名规则：什么叫"几层"网络？

有一个容易混淆的地方：**输入层不算层数**。

- 1 个隐藏层 + 1 个输出层 = "2 层网络"
- 2 个隐藏层 + 1 个输出层 = "3 层网络"

**记忆技巧：** 只有"干了活"的层才算数。输入层只是收数据，没干活，所以不算。

## 原理解析

### 信号是怎么传递的？

数据在神经网络中的流动分三步：

```
第 1 步：计算加权和 → a = w·x + b
第 2 步：通过激活函数 → z = h(a)
第 3 步：传给下一层
```

**这到底意味着什么？**

想象你在参加一个评分比赛：
- 评委给 5 个维度打分（x1 到 x5）→ **这是输入**
- 每个维度有不同的重要性（w1 到 w5）→ **这是权重**
- 你有个基础分（b）→ **这是偏置**
- 把加权分加起来 → **这是加权和 a**
- 用一个规则转换分数（比如满分 100 分）→ **这是激活函数 h(a)**
- 得到最终分数 → **这是输出 z**

### 为什么叫"神经"网络？

因为它的灵感来自生物大脑。

**生物神经元：** 接收多个信号 → 信号总和超过阈值 → "放电"（输出信号）

**人工神经元：** 接收多个数字 → 加权和超过某个值 → 输出一个数字

本质上是一回事，只是人工版用数字代替了生物电信号。

## 代码实战

下面我们用代码实现一个最简单的神经网络。

这个网络结构：2 个输入 → 3 个隐藏节点 → 2 个输出。

### 第 1 步：定义激活函数

```python
import numpy as np

def sigmoid(x):
    """Sigmoid 函数：把任意数字压缩到 0~1 之间"""
    return 1 / (1 + np.exp(-x))

def identity_function(x):
    """恒等函数：输入什么输出什么"""
    return x
```

**解读：**
- `sigmoid` 是我们后面要详细学的激活函数，现在只需要知道它能把数字限制在 0 到 1 之间。
- `identity_function` 最简单，输入 3 就输出 3，什么都不做。

### 第 2 步：定义网络参数

```python
# 输入数据：2 个数字
x = np.array([1.0, 0.5])

# 第 1 层的权重：2 行 3 列
# 2 行 = 2 个输入，3 列 = 3 个隐藏节点
W1 = np.array([
    [0.1, 0.3, 0.5],   # 输入 1 到 3 个隐藏节点的权重
    [0.2, 0.4, 0.6]    # 输入 2 到 3 个隐藏节点的权重
])

# 第 1 层的偏置：3 个数字（对应 3 个隐藏节点）
b1 = np.array([0.1, 0.2, 0.3])

# 第 2 层的权重：3 行 2 列
# 3 行 = 3 个隐藏节点，2 列 = 2 个输出
W2 = np.array([
    [0.1, 0.4],   # 隐藏节点 1 到 2 个输出的权重
    [0.2, 0.5],   # 隐藏节点 2 到 2 个输出的权重
    [0.3, 0.6]    # 隐藏节点 3 到 2 个输出的权重
])

# 第 2 层的偏置：2 个数字（对应 2 个输出）
b2 = np.array([0.1, 0.2])
```

**解读：**
- 权重的行数 = 上一层的节点数，列数 = 这一层的节点数。
- 偏置的个数 = 这一层的节点数。
- 这些数字现在是随便设的，后面会学怎么自动找到好的权重。

### 第 3 步：前向传播（数据从输入到输出）

```python
# === 输入层 → 第 1 层（隐藏层）===
a1 = np.dot(x, W1) + b1    # 计算加权和
z1 = sigmoid(a1)            # 通过激活函数

# === 第 1 层 → 第 2 层（输出层）===
a2 = np.dot(z1, W2) + b2   # 计算加权和
y = identity_function(a2)   # 输出层的激活函数

print("输出结果:", y)
```

**逐行解读：**
1. `np.dot(x, W1)` 是矩阵乘法。x 是 `[1.0, 0.5]`（1×2），W1 是 2×3。结果是 1×3 的数组。
2. `+ b1` 把偏置加上去。
3. `a1` 就是加权和，还没经过激活函数。
4. `sigmoid(a1)` 把 a1 的每个数字压缩到 0~1 之间，得到 z1。
5. 第 2 层用 z1 作为输入，重复同样的步骤。
6. 输出层用 `identity_function`，输入什么输出什么。

### 完整代码

把上面的代码拼在一起，就是一个能运行的神经网络：

```python
import numpy as np

def sigmoid(x):
    return 1 / (1 + np.exp(-x))

def identity_function(x):
    return x

# 输入和参数
x = np.array([1.0, 0.5])
W1 = np.array([[0.1, 0.3, 0.5], [0.2, 0.4, 0.6]])
b1 = np.array([0.1, 0.2, 0.3])
W2 = np.array([[0.1, 0.4], [0.2, 0.5], [0.3, 0.6]])
b2 = np.array([0.1, 0.2])

# 前向传播
a1 = np.dot(x, W1) + b1
z1 = sigmoid(a1)
a2 = np.dot(z1, W2) + b2
y = identity_function(a2)

print("输出结果:", y)
```

运行后你会看到类似这样的输出：
```
输出结果: [0.31682708 0.69627909]
```

这两个数字就是网络的输出。现在它们没有实际意义，因为权重是随便设的。等网络"学习"之后，这两个数字就会代表有意义的信息（比如"这是数字 3 的概率"）。

## 避坑指南

1. **矩阵维度要对上** - 这是最常见的错误。记住：权重的行数 = 输入的个数，列数 = 输出的个数。如果报错 `ValueError: shapes not aligned`，检查维度。

2. **输入层不算层数** - "2 层网络" = 1 个隐藏层 + 1 个输出层。输入层只是收数据，不算。

3. **权重现在是随便设的** - 这个网络还不会做任何有用的事。第 4 章会学怎么让网络自己找到好的权重。
