# 从零开始深度学习 - 学习笔记

## 项目介绍

本项目是《从零开始深度学习》(ゼロから作る Deep Learning) 的配套学习笔记。所有笔记专为 **零基础开发者** 编写，用纯中文、通俗的语言解释深度学习的核心概念，配合从零手写的代码示例，帮助你真正理解神经网络的工作原理。

## 适合谁学？

- 会写Python基本代码，但从未接触过深度学习的开发者
- 数学基础是高中水平（了解导数概念即可）
- 想自己动手实现神经网络，而不是只会调用框架的开发者

**零基础学习指南**：`docs/零基础学习指南.md` - 详细的3个月学习路径规划

## 环境准备

### 需要安装的软件
1. Python 3.6 或更高版本
2. NumPy（用于数组运算）
3. Matplotlib（用于画图）

```bash

# 2. 运行代码

# 3. 启动 Jupyter

```

## 如何使用笔记？

每篇笔记包含六个部分：
1. **学习目标** - 本节要学什么
2. **核心概念** - 用大白话解释，不用公式
3. **原理解析** - 深入理解背后的逻辑
4. **代码实战** - 从零手写，逐行解析
5. **避坑指南** - 新手常犯错误
6. **课后思考** - 动手修改代码的练习

### 学习建议

1. **先跑代码，再看理论** - 看到实际输出更容易理解
2. **动手修改参数** - 理解概念的最好方式是实验
3. **不必深究数学** - 高中数学足够，重点理解直觉
4. **用类比理解** - 项目大量使用类比解释概念

> **建议：** 不要只看不练。每看完一节，一定要动手运行代码，并尝试修改参数看看会发生什么。
